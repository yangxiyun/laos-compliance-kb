# 外籍员工-证件办理费用

## 核心要点
- 外籍员工三大必需证件：工作许可证、居留许可证、签证
- 年度总成本约188美元/人（不含存档费）
- 存档费20美元/次，为一次性费用
- 经济特区外籍员工个人所得税仅5%
- 老挝社保雇主承担6%，员工承担5.5%

## 详细内容

### 证件办理费用（年度）
**工作许可证**
- 费用：1,200,000老普 = 61美元/人/年
- 必需文件，合法工作的基础

**居留许可证**
- 费用：1,300,000老普 = 66美元/人/年
- 合法居留的必要证件

**签证**
- 费用：1,200,000老普 = 61美元/人/年
- 出入境必需

**存档费**
- 费用：400,000老普 = 20美元/次
- 一次性费用

### 社会保险
**缴费基数**
- 最高限额：4,500,000老普/月

**缴费比例**
- 雇主承担：6%
- 员工承担：5.5%

### 税务待遇
- 经济特区内外籍员工个人所得税：5%（相比普通地区有优惠）

## 实务注意事项
- 三证必须同时办理和维护，缺一不可
- 证件到期前需提前续办，避免影响工作和居留
- 存档费为一次性费用，首次办理时需计入成本
- 社保缴费基数有上限，高薪员工实际社保成本相对较低
- 汇率变动会影响实际美元成本

## 关键条款摘录
**证件费用标准**
- "工作许可证: 套管 1,200,000 = 61 美元/人/年"
- "居留许可证: 套管 1,300,000 = 66 美元/人/年"
- "签证: 套管 1,200,000 = 61 美元/人/年"
- "注: 已存档费用: 套管 400,000 = 20 美元/1次"

**社保规定**
- "最高限额: 4,500,000老普/月"
- "雇主: 6%，员工: 5.5%"
