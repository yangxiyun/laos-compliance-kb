# 出口退税管理规定与服务举措

## 一、概述

本页涵盖中国出口退税的**管理规定（项目68-71）**和**便捷服务举措（项目101-107）**：

| 类别 | 项目 | 核心内容 |
|---|---|---|
| 管理规定 | 68 | 出口退（免）税备案 |
| 管理规定 | 69 | 出口退税申报办理（申报期限、申报资料） |
| 管理规定 | 70 | 出口退（免）税备案单证管理 |
| 管理规定 | 71 | 完善出口退（免）税收汇管理 |
| 服务举措 | 101 | 出口退（免）税无纸化申报 |
| 服务举措 | 102 | 简化代办退税备案流程 |
| 服务举措 | 103 | 推行出口退（免）税实地核查"容缺办理" |
| 服务举措 | 104 | 简便出口退（免）税办理方式（电子证明、非接触办理） |
| 服务举措 | 105 | 增加出口退（免）税便捷服务 |
| 服务举措 | 106 | 缩减出口退（免）税办理时间 |
| 服务举措 | 107 | 出口退税率查询 |

## 二、项目68：出口退（免）税备案

### 基本规则

1. 纳税人可在**首次申报出口退（免）税时**，向主管税务机关办理出口退（免）税备案手续；
2. 备案时应真实、完整填写并报送：
   - **《出口退（免）税备案表》**（"退税开户银行账号"**必须填写办理税务登记时向主管税务机关报备的银行账号之一**）；
   - 主管税务机关要求提供的其他资料；
3. 办理备案变更时，在《出口退（免）税备案表》中**仅需填报变更的内容**。

### 实务要点

- 备案是办理出口退（免）税的**前置条件**；
- 退税开户银行账号必须已向税务机关报备；
- 变更备案时只填变更项，无需重复整表。

## 三、项目69：出口退税申报办理

### 1. 申报期限（重要）

#### 基本规则

- 纳税人出口并按会计规定做销售的货物，应在做销售的次月进行**增值税纳税申报**；
- 纳税人在货物报关出口之日（以出口货物报关单上载明的**"出口日期"**为准）次月增值税纳税申报期内，即可向主管税务机关申报办理出口退（免）税；
- **最晚不迟于次年4月30日前**的任何一个月增值税纳税申报期，均可申报办理出口退（免）税。

#### 外贸企业的特例

对于外贸企业，经主管税务机关同意后，也可在**增值税纳税申报期以外的其他时间**办理出口退（免）税申报。

#### 2020年重大政策变化

自**2020年1月20日起**（财税总局2020年第2号），纳税人**未在规定期限内申报出口退（免）税的**，待收齐退（免）税凭证及相关电子信息后，**仍可申报办理出口退（免）税**。

### 2. 申报资料

#### 生产企业（免抵退税）

应提供申报电子数据及下列表单资料：

1. **《免抵退税申报汇总表》**；
2. **《生产企业出口货物劳务免抵退税申报明细表》**；
3. **出口货物报关单、出口发票**等凭证。

**委托出口的**：还应提供**受托方主管税务机关签发的代理出口货物证明**。

**消费税退税（如需）**：还需提供：

- 《生产企业出口非自产货物消费税退税申报表》；
- 消费税专用缴款书或分割单、海关进口消费税专用缴款书、委托加工收回应税消费品的代扣代收税款凭证等。

#### 外贸企业（免退税）

应提供申报电子数据及下列表单资料：

1. **《外贸企业出口退税进货明细申报表》**；
2. **《外贸企业出口退税出口明细申报表》**；
3. **出口货物报关单、增值税专用发票、海关进口增值税专用缴款书**等凭证。

**委托出口的**：还应提供受托方主管税务机关签发的代理出口货物证明。

**消费税退税（如需）**：还应提供消费税专用缴款书或分割单、海关进口消费税专用缴款书等凭证。

## 四、项目70：出口退（免）税备案单证管理

### 基本要求

纳税人应在**申报出口退（免）税后15日内**，将下列备案单证妥善留存：

- 按照申报退（免）税的时间顺序；
- 制作**出口退（免）税备案单证目录**；
- 注明单证存放方式；
- 以备税务机关核查。

### 3类必备单证

#### 1. 出口企业的购销合同

包括：

- 出口合同；
- 外贸综合服务合同；
- 外贸企业购货合同；
- 生产企业收购非自产货物出口的购货合同等。

#### 2. 出口货物的运输单据

包括：

- 海运提单、航空运单、铁路运单、货物承运单据、邮政收据等**承运人出具的货物单据**；
- 出口企业承付运费的**国内运输发票**；
- 出口企业承付费用的**国际货物运输代理服务费发票**等。

#### 3. 出口企业委托其他单位报关的单据

包括：

- 委托报关协议；
- 受托报关单位为其开具的代理报关服务费发票等。

### 替代方案

纳税人无法取得上述单证的，可用**具有相似内容或作用的其他资料**进行单证备案。

### 保存规则

- 除另有规定外，备案单证由出口企业**存放和保管**；
- **不得擅自损毁**；
- 保存期为**5年**；
- 纳税人可以自行选择**纸质化、影像化或者数字化**方式留存保管；
- 选择纸质化方式的，还需在出口退（免）税备案单证目录中**注明备案单证的存放地点**；
- 税务机关按规定查验备案单证时，纳税人按要求将影像化或者数字化备案单证转换为纸质化备案单证以供查验的，应在纸质化单证上**加盖企业印章并签字声明与原数据一致**。

## 五、项目71：完善出口退（免）税收汇管理

### 1. 收汇基本要求

纳税人申报退（免）税的出口货物，应当在**出口退（免）税申报期截止之日前收汇**。

### 2. 视同收汇的情形

未在规定期限内收汇，但符合**《视同收汇原因及举证材料清单》**所列原因的，纳税人留存**《出口货物收汇情况表》**及举证材料，即可**视同收汇**。

### 3. 合同约定的收汇特例

因出口合同约定全部收汇最终日期在退（免）税申报期截止之日后的，应当**在合同约定收汇日期前完成收汇**。

### 4. 收汇材料简化（重要）

**除以下3种情形外**，纳税人申报出口退（免）税时，**无需报送收汇材料**，留存举证材料备查即可：

1. 出口退（免）税管理类别为**四类**的纳税人；
2. 纳税人在退（免）税申报期**截止之日后**申报出口货物退（免）税；
3. 纳税人被税务机关发现收汇材料为**虚假或冒用**。

税务机关按规定需要查验收汇情况的，纳税人应当按照税务机关要求报送收汇材料。

## 六、项目101：出口退（免）税无纸化申报

### 适用条件（4个条件）

符合以下条件的出口企业，均可采用无纸化方式申报办理出口退（免）税事项：

1. 自愿申请开展出口退（免）税无纸化管理，且向主管税务机关**承诺能够按规定将有关申报资料留存企业备案**；
2. 出口退（免）税企业分类管理类别为**一类、二类、三类**（即不包括四类）；
3. 有**税控数字签名证书**或主管税务机关认可的其他数字签名证书；
4. 能够按规定报送经**数字签名后**的出口退（免）税**全部申报资料**的电子数据。

### 无纸化的好处

企业通过提供**数字签名证书签名后的正式申报电子数据**，即可申报办理出口退（免）税相关事项：

- 无需报送纸质申报表和纸质凭证；
- 原规定向主管税务机关报送的纸质凭证**留存备查**。

### 特殊情形

出口企业和其他单位**视同出口货物**，在提供退（免）税申报资料时，存在无法生成电子数据情形的，**不影响该企业申请成为无纸化申报试点企业**，该项视同出口业务仍可按现行规定报送纸质申报资料办理退（免）税。

## 七、项目102：简化代办退税备案流程

### 政策内容

外贸综合服务企业在生产企业办理委托代办退税备案后，留存以下资料即可为该生产企业申报代办退税，**无需报送《代办退税情况备案表》和企业代办退税风险管控制度**：

1. 与生产企业签订的外贸综合服务合同（协议）；
2. 每户委托代办退税生产企业的《代办退税情况备案表》；
3. 外贸综合服务企业代办退税风险管控制度、内部风险管控信息系统建设及应用情况。

### 变更处理

生产企业办理委托代办退税备案**变更**后，外贸综合服务企业将**变更后的《代办退税情况备案表》留存备查即可**，无需重新报送该表。

## 八、项目103：出口退（免）税实地核查"容缺办理"

### 政策内容

对于纳税人按照现行规定**需实地核查**通过方可办理的以下出口退（免）税：

- **首次申报**的出口退（免）税；
- **变更退（免）税办法后首次申报**的出口退（免）税。

税务机关经审核**未发现涉嫌骗税等疑点**或者**已排除涉嫌骗税等疑点**的，应按照"**容缺办理**"的原则办理退（免）税：

- 在该纳税人累计申报的应退（免）税额**未超过限额前**，可先行按规定审核办理退（免）税再进行实地核查；
- 在该纳税人累计申报的应退（免）税额**超过限额后**，超过限额的部分需待**实地核查通过后**再行办理退（免）税。

### 限额标准

| 企业类型 | 累计申报应退（免）税额限额 |
|---|---|
| **外贸企业**（含外贸综合服务企业自营出口业务） | **100万元** |
| **生产企业**（含生产企业委托代办退税业务） | **200万元** |
| **代办退税的外贸综合服务企业** | **100万元** |

### 适用范围

适用实地核查"容缺办理"的出口退（免）税业务，**既包括**：

- 企业出口货物；
- 视同出口货物申报的出口退（免）税；
- 企业对外提供加工修理修配劳务；
- 发生**零税率跨境应税行为**申报的出口退（免）税。

## 九、项目104：简便出口退（免）税办理方式

### 1. 推广出口退（免）税证明电子化开具和使用

纳税人申请开具以下**6类证明**的，税务机关为其开具**电子证明**：

1. 《代理出口货物证明》；
2. 《代理进口货物证明》；
3. 《委托出口货物证明》；
4. 《出口货物转内销证明》；
5. 《中标证明通知书》；
6. 《来料加工免税证明》。

通过**电子税务局、国际贸易"单一窗口"**等网上渠道向纳税人反馈。纳税人申报办理出口退（免）税相关涉税事项时，**仅需填报上述电子证明编号等信息**，无需另行报送证明的纸质件和电子件。

**特别简化**：纳税人申请开具《中标证明通知书》时，**无需再报送**中标企业所在地主管税务机关的名称、地址、邮政编码。

### 2. 推广出口退（免）税事项"非接触"办理

纳税人申请办理出口退（免）税备案、证明开具及退（免）税申报等事项时：

- 按照现行规定需要现场报送的纸质表单资料；
- 可选择通过网上渠道，以**影像化或者数字化方式**提交；
- 纳税人通过网上渠道提交相关电子数据、影像化或者数字化表单资料后，即可完成相关出口退（免）税事项的申请。

## 十、项目105：增加出口退（免）税便捷服务

### 3种免费申报渠道

税务部门提供了**3种免费申报渠道**供纳税人选用：

1. **电子税务局**；
2. **标准版国际贸易"单一窗口"**；
3. **出口退税离线申报工具**。

### 5项便捷服务功能

纳税人可通过上述申报渠道，提出以下5类申请：

1. **出口退（免）税备案撤回**；
2. 已办结退税的出口货物免退税申报，发现申报数据有误而作**申报调整**；
3. 将申请出口退税的**增值税专用发票、海关进口增值税专用缴款书用途改为申报抵扣**；
4. 出口退（免）税**相关证明作废**；
5. **进料加工计划分配率调整**。

## 十一、项目106：缩减出口退（免）税办理时间

### 办理时限（重要）

确保出口企业**正常**出口退税平均办理时间：

| 企业类型 | 正常办理时间 |
|---|---|
| 全部正常出口退税 | **6个工作日**内 |
| **一类、二类出口企业** | **3个工作日**内 |

一类、二类出口企业的3个工作日政策**延续实施至2024年底**。

## 十二、项目107：出口退税率查询

### 查询方式

税务总局在官网"纳税服务"栏目"我要查"模块设置了"**出口退税率查询**"功能，为企业提供出口货物的出口退税率查询服务。

**网址**：http://hd.chinatax.gov.cn/nszx2023/cktslcx2023.html

## 情景问答

### Q1：新办的自行车生产企业准备出口东南亚，出口货物退（免）税申报程序的要求是什么？

**答**：

1. **首次申报时办理备案**：向主管税务机关办理出口退（免）税备案（项目68），填报《出口退（免）税备案表》；
2. **按申报期申报**：在各增值税纳税申报期内，收齐退（免）税凭证及相关电子信息后，向主管税务机关申报办理出口货物增值税**免抵退税**（项目69）；
3. **留存备案单证**：申报出口退（免）税后**15日内**，将购销合同、运输单据、委托报关单据等**备案单证**妥善留存（项目70），保存期**5年**；
4. **收汇管理**：在**退（免）税申报期截止之日前**完成收汇（项目71）；
5. **申报期限**：最晚不迟于**次年4月30日前**的任何一个月增值税纳税申报期。

### Q2：新办的外贸企业如何办理出口退（免）税备案？

**答**：出口企业或其他单位应于**首次申报出口退（免）税时**，向主管税务机关提供以下资料办理出口退（免）税备案手续：

1. 内容填写真实、完整的**《出口退（免）税备案表》**，其中"退税开户银行账号"须从向税务机关报告的银行账号中选择一个填报；
2. 未办理备案登记发生委托出口业务的生产企业提供**委托代理出口协议**；
3. 主管税务机关要求提供的其他资料。

### Q3：外贸企业留存备案单证时，是否可以直接采用影像化的留存方式？

**答**：**可以**。外贸企业可根据自身实际，**自行选择纸质化、影像化或者数字化方式**，留存保管出口退（免）税备案单证。

**注意**：

- 选择纸质化方式的，还需在出口退（免）税备案单证目录中**注明备案单证的存放地点**；
- 税务机关按规定查验备案单证时，纳税人按要求将影像化或数字化备案单证转换为纸质化备案单证以供查验的，应在纸质化单证上**加盖企业印章并签字声明与原数据一致**。

### Q4：外贸企业留存的海运提单等备案单证资料多是影像化资料，是否符合规定？

**答**：**符合**。按规定可采用影像化方式留存备案单证。保存期5年，以备税务机关核查。

### Q5：跨境应税行为适用增值税零税率出口退（免）税业务是否适用实地核查"容缺办理"？

**答**：**可以**。适用实地核查"容缺办理"的出口退（免）税业务，**既包括**：

- 企业出口货物；
- 视同出口货物申报的出口退（免）税；
- 企业对外提供加工修理修配劳务；
- 发生**零税率跨境应税行为**申报的出口退（免）税。

### Q6：已办结退税的出口货物免退税申报，发现申报数据有误，能否调整？

**答**：**可以调整**。根据项目105，纳税人可通过3种免费申报渠道（电子税务局、标准版国际贸易"单一窗口"、出口退税离线申报工具），对**已办结退税的出口货物免退税申报，发现申报数据有误而作申报调整**。

### Q7：一类出口企业的出口退税办理通常需要多长时间？

**答**：

- **正常**出口退税平均办理时间：**6个工作日**内；
- **一类、二类**出口企业正常出口退（免）税平均办理时间：**3个工作日**内；
- 一类、二类企业的3个工作日政策延续实施至**2024年底**。

## 用户操作清单

### 服务中资客户时的实务要点

本页内容主要面向中国境内的出口企业操作，对ZTL的老挝业务直接相关度较低。但在以下场景可能涉及：

#### 场景1：客户的中国母公司/关联方向老挝出口货物

1. 提示客户中国端的备案、申报、单证管理、收汇要求；
2. 协助协调**两端资料一致性**：
   - 中国出口单证与老挝进口单证的金额、货物清单、日期一致；
   - 避免因两端数据不一致被质疑；
3. 若客户选择**无纸化申报**（项目101），所有纸质凭证留存企业备查，ZTL协助整理老挝端的配套资料；
4. 若客户是新办企业首次申报，提示**容缺办理**（项目103）：
   - 外贸企业限额100万元；
   - 生产企业限额200万元；
   - 未超过限额可先办理退税再实地核查。

#### 场景2：客户的合同设计涉及"合同约定全部收汇最终日期在退税申报期截止日之后"

1. 这种情形常见于大型EPC工程、长期贸易合同；
2. 提示客户中国端出口退税的**特殊收汇规则**：
   - 应当在合同约定收汇日期前完成收汇；
   - 否则可能影响中国端退税；
3. 协调老挝端付款时点的合理安排。

### 关键管理规定速查

| 管理事项 | 关键要求 |
|---|---|
| 备案时点 | 首次申报时办理 |
| 备案变更 | 仅填变更项 |
| 最晚申报期 | 次年4月30日前 |
| 2020年后 | 未按期申报的仍可申报 |
| 备案单证留存 | 申报后15日内 |
| 备案单证保存期 | 5年 |
| 收汇时点 | 申报期截止日前 |
| 正常退税办理 | 6个工作日内 |
| 一类二类企业 | 3个工作日内 |
| 实地核查容缺办理 | 外贸100万/生产200万 |

### 常见陷阱

1. **备案表"退税开户银行账号"错填**：必须是已向税务机关报备的银行账号之一，填写未报备账号会导致备案失败；
2. **备案单证15日内留存要求**：申报后15日内必须完成留存，逾期可能被质疑；
3. **备案单证保存期5年**：不得擅自损毁；
4. **收汇材料的3种例外情形**：四类纳税人、截止日后申报、虚假冒用——这3种情况需额外报送收汇材料；
5. **无纸化申报的四类企业除外**：四类企业不能享受无纸化申报；
6. **容缺办理的限额累计**：外贸100万、生产200万，超过后超过部分需实地核查通过；
7. **申报渠道的3种选择**：电子税务局、单一窗口、离线工具，三者功能略有差异。
