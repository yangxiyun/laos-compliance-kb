# 境外所得税收抵免-直接抵免与间接抵免

## 一、概述

中国居民企业应当就其来源于中国境内、境外的所得缴纳企业所得税。对于企业在境外缴纳的所得税性质的税款，中国实行**税收抵免政策**，即允许企业用境外已缴税款抵免其境内外所得应纳税总额。

税收抵免分为：

- **直接抵免**：企业就来源于境外的所得直接缴纳或被扣缴的所得税；
- **间接抵免**：企业取得境外子公司股息时，该股息间接负担的境外所得税；
- **饶让抵免**（特殊情形）：根据税收协定，境外减免税视同已缴税款抵免；
- **简易办法抵免**：特定情形下适用定率计算或"白名单"简化办理。

## 二、境外所得的确认

### 1. 所得来源地判定

| 所得类型 | 来源地判定规则 |
|---|---|
| 销售货物所得 | 交易活动发生地 |
| 提供劳务所得 | 劳务发生地 |
| 转让不动产所得 | 不动产所在地 |
| 转让动产所得 | 转让动产的企业或机构、场所所在地 |
| 转让权益性投资资产所得 | 被投资企业所在地 |
| 股息、红利等权益性投资所得 | 分配所得的企业所在地 |
| 利息、租金、特许权使用费所得 | 负担、支付所得的企业或机构、场所所在地 |
| 其他所得 | 由国务院财政、税务主管部门确定 |

### 2. 所得实现年度

- **股息、红利所得**：若实际收到所得的日期与境外被投资方作出利润分配决定的日期不在同一纳税年度的，应按被投资方作出利润分配日所在的纳税年度确认境外所得；
- **利息、租金、特许权使用费、转让财产所得**：若未能在合同约定的付款日期当年收到，仍应按合同约定付款日期所属的纳税年度确认境外所得。

### 3. 所得的计算（重要）

- **直接抵免时**：境外所得应为将该项境外所得直接缴纳的境外所得税额**还原计算后的境外税前所得**；
- **间接抵免时**：境外股息、红利所得应为境外股息、红利**税后净所得与就该项所得直接缴纳和间接负担的税额之和**。

**举例**：境内企业取得来自A国特许权使用费所得56万元（税后净额），在A国按30%税率缴纳所得税。还原后境外应纳税所得额 = 56 ÷ (1 - 30%) = **80万元**。

## 三、直接抵免

### 适用情形

- 企业就来源于境外的营业利润所得在境外所缴纳的企业所得税；
- 就来源于或发生于境外的股息、红利等权益性投资所得、利息、租金、特许权使用费、财产转让等所得在境外被**源泉扣缴的预提所得税**。

### 抵免限额计算公式

```
抵免限额 = 中国境内、境外所得依照企业所得税法规定计算的应纳税总额 
         × 来源于某国（地区）的应纳税所得额 
         ÷ 中国境内、境外应纳税所得总额
```

### 抵免方式选择（重要）

企业可以选择以下两种方式之一：

1. **分国（地区）不分项**：按国别分别计算抵免限额；
2. **不分国（地区）不分项**：不按国别汇总计算抵免限额。

**关键规则**：

- 上述方式一经选择，**5年内不得改变**；
- 若改变方式，以前年度未抵免完的余额，可在税法规定结转的剩余年限内，按新方式计算的抵免限额中继续结转抵免。

### 结转年限

- 超过抵免限额的部分，可以在**以后5个年度**内，用每年度抵免限额抵免当年应抵税额后的余额进行抵补。

### 可抵免的境外所得税认定（4个要件）

符合以下条件的税款，均可确定为可抵免的境外所得税税额：

1. 企业来源于中国境外的所得，依照中国境外税收法律及相关规定计算而缴纳的税额；
2. **缴纳的属于企业所得税性质的税额，而不拘泥于名称**（如法人所得税、利得税、公司所得税等；老挝的"利润税/Profit Tax"属于此类）；
3. 限于企业应当缴纳且**已实际缴纳**的税额；
4. 如果是税收协定非适用所得税项目，或来自非协定国家的所得，无法判定是否属于对企业征收的所得税税额的，应层报国家税务总局裁定。

### 不可抵免的境外所得税（6种情形）

1. 按照境外所得税法律及相关规定属于**错缴或错征**的境外所得税税款；
2. 按照税收协定规定**不应征收**的境外所得税税款；
3. 因少缴或迟缴境外所得税而追加的**利息、滞纳金或罚款**；
4. 境外所得税纳税人或者其利害关系人从境外征税主体得到实际**返还或补偿**的境外所得税税款；
5. 按照中国企业所得税法及其实施条例规定，已经免征中国企业所得税的境外所得负担的境外所得税税款；
6. 按照国务院财政、税务主管部门有关规定已经从企业境外应纳税所得额中扣除的境外所得税税款。

### 凭证要求与追溯抵免

- 企业抵免企业所得税税额时，应当提供中国境外税务机关出具的税款所属年度的有关纳税凭证；
- 收到某一纳税年度的境外所得已纳税凭证凡是**迟于次年5月31日汇算清缴终止日**的，可以对该所得境外税额抵免追溯计算。

## 四、间接抵免

### 适用情形

取得境外股息、红利等权益性投资收益且满足相关要求的居民企业。

### 基本规则

- 境外子公司在境外缴纳的企业所得税中，**属于分配给中国母公司股息对应的部分**，可由中国母公司在中国进行间接抵免；
- 间接抵免限层级：一般为**五层持股**（最低一层持股比例达20%）。

### 持股比例要求

- 直接持有或间接持有外国企业**20%以上股份**（含）的条件。

## 五、境外分支机构应纳税所得额计算特殊规则

若中资企业在老挝设立**分公司（不具有独立纳税地位的分支机构）**：

1. 境外分支机构取得的各项所得，**不论是否汇回境内**，均应当计入所属年度的企业应纳税所得额；
2. 分支机构不具有分配利润职能；
3. **亏损限制**：分国（地区）抵免法下，境外分支机构的亏损，不得抵减其境内或他国（地区）的应纳税所得额，但可以用同一国家（地区）其他项目或以后年度的所得按规定弥补；
4. 企业在同一纳税年度的境内外所得加总为正数的，境外分支机构发生的亏损因结转弥补限制而未予弥补的部分，今后在该分支机构的**结转弥补期限不受5年期限制**。

## 六、简易办法抵免

企业从境外取得营业利润所得以及符合境外税额间接抵免条件的股息所得，凡就该所得**在所得来源国（地区）的实际有效税率高于12.5%**的，可直接以境外应纳税所得额的**12.5%**作为抵免限额。

对以下国家（地区）为来源地的境外所得实行"白名单"简易抵免：法定税率明显高于中国且实际有效税率不低于中国的国家（地区）。

## 七、抵免办理的申报要求

### 申报表

取得境外所得的纳税人应当在办理企业所得税年度申报时向主管税务机关填报：

1. 《境外所得税收抵免明细表》（A108000）；
2. 《境外所得纳税调整后所得明细表》（A108010）；
3. 《境外分支机构弥补亏损明细表》（A108020）；
4. 《跨年度结转抵免境外所得税明细表》（A108030）。

### 应提交的书面资料

**不同类型的境外所得分别提供**：

- **境外分支机构营业利润**：分支机构会计报表、具有资质的机构出具的分支机构审计报告；
- **境外利息、租金、特许权使用费、转让财产所得**：依照中国税法计算应纳税额的资料及计算过程。

**申请饶让抵免的还需提供**：本企业及其直接或间接控制的外国企业在境外所获免税及减税的依据及证明或有关审计报告披露的优惠政策复印件。

### 留存备查资料

- 与境外所得相关的完税证明或纳税凭证（原件或复印件）；
- 境外分支机构所得应纳税额的计算过程及说明资料；
- 取得境外股息、红利所得需提供集团组织架构图、被投资公司章程复印件、境外企业有权决定利润分配的机构作出的决定书等；
- 取得境外利息、租金、特许权使用费、转让财产等所得项目合同复印件等；
- 非中文资料需同时提交中文译本复印件。

## 情景问答

### Q1：境外所得抵免是不是只有签订税收协定的国家才能享受？

**答**：**不是**。企业可以抵免的境外所得税并不限于在已与中国签订避免双重征税协定的国家（地区）已缴纳的所得税。因此，境外投资企业在还没有与中国政府签订避免双重征税协定的国家（地区）已缴纳的所得税，仍然可以按照相关国内法规定在计算中国应纳税额中抵免。

### Q2：境外缴纳税款的税种名称与中国不同（如老挝叫"利润税"而非"企业所得税"），是否属于可抵免的境外所得税？

**答**：可抵免的境外所得税**不拘泥于名称**，判定标准是：

1. 依照境外税收法律及相关规定计算缴纳；
2. 属于企业所得税性质（对企业净所得征税）；
3. 实际缴纳而非错缴；
4. 不能是税收协定规定不应征收的税款。

因此，老挝征收的**利润税（Profit Tax）**属于企业所得税性质的税款，可以在中国申请抵免。类似地，香港的利得税、哈萨克斯坦的超额利润税（见国家税务总局公告2019年第1号）等均属可抵免范围。

### Q3：境外缴纳的所得税都能回国申报抵免吗？有哪些特殊情况？

**答**：以下情形不能抵免：

1. 按境外所得税法律属于**错缴或错征**的；
2. 按税收协定规定**不应征收**的（应向境外主管税务机关申请退税）；
3. 因少缴或迟缴而追加的**利息、滞纳金或罚款**；
4. 从境外征税主体实际**返还或补偿**的税款；
5. 已经**免征中国企业所得税**的境外所得负担的境外税款；
6. 已经从境外应纳税所得额中**扣除**的境外所得税税款。

**特别提示**：如果境外适用税率高于协定限定税率，多缴纳的税款应向境外主管税务机关申请退税，不能在中国抵免。

### Q4：老挝子公司股权转让被老挝扣缴了预提所得税，能否在中国抵免？

**答**：可以。此类来源于或发生于境外的财产转让所得，如在境外被源泉扣缴预提所得税，**可适用直接抵免**：

- 按规定计算抵免限额；
- 未超过限额的部分直接从境内当期应纳税额中抵免；
- 超过限额的部分，允许从次年起在连续5个纳税年度内结转抵补。

### Q5：若老挝征税税率高于中老协定限制税率，多缴的部分如何处理？

**答**：该笔多缴税款**不能在境内进行税收抵免**。企业应按照税收协定的规定向**老挝税务局申请退还**不应征收的税款。需要注意的是，企业境外所得在来源国纳税时适用税率高于税收协定限定税率所多缴纳的所得税税额，也属于上述申请退还的范围。

## 用户操作清单

### 服务中资母公司时（尽调与合规）

1. **确认境外子公司税负**：收集老挝子公司的利润税申报表、完税凭证（税单）；
2. **收集抵免资料**：
   - 老挝子公司年度审计报告；
   - 老挝税务局出具的完税证明（若能取得）；
   - 分配决议、银行付款凭证（股息抵免用）；
   - 集团组织架构图（间接抵免用）；
3. **提示抵免方式选择**：告知客户"分国不分项" vs "不分国不分项"一经选择5年内不得改变；
4. **提示凭证要求**：跨年度纳税凭证可追溯抵免（迟于次年5月31日到账的凭证允许追溯）；
5. **准备中文译本**：非中文资料需同时提交中文译本复印件。

### 实务中的典型陷阱

1. **老挝SEZ减免税**：因中老协定无饶让抵免条款，老挝减免的税款**不能**在中国抵免，仅实际缴纳的部分可抵免；
2. **工程项目承包**：境外承包工程若采用总分包方式，需使用《境外承包工程项目完税凭证分割单》（总分包方式）或（联合体方式）分配完税凭证；
3. **股权转让**：老挝扣缴的股权转让预提所得税属于可直接抵免范围；
4. **利息税率差异**：若实际缴纳税率高于协定税率（如某协定规定不超过7%但实际按10%扣缴），超额部分应向老挝申请退税，不能在中国抵免。

## 备注

- 中老税收协定**无饶让抵免条款**，是中资企业在老挝投资面临的重要税收成本因素，需在投资架构设计阶段予以充分考虑；
- 若未来中老双方重新谈签协定并加入饶让抵免条款，该情况将发生重大改变，应持续关注中国税务总局发布的税收协定更新信息；
- ZTL在服务中资客户的母公司端（中国本土）税务时，应与境内税务师事务所协同作业，确保凭证要求、报表填报符合国家税务总局最新规定。
