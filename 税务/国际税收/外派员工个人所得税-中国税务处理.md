# 外派员工个人所得税-中国税务处理

## 一、概述

中资企业（派出单位）向老挝派遣员工，其员工（居民个人）在老挝工作期间取得的工资薪金所得，涉及**中老两国的个人所得税义务**：

1. **中国端**：居民个人应就其全球所得在中国纳税，派出单位承担代扣代缴义务，员工次年3-6月需办理年度自行申报及境外税收抵免；
2. **老挝端**：员工在老挝产生纳税义务，按老挝PIT规则由老挝雇主（或实际支付者）代扣代缴。

本页聚焦**中国端的税务处理**，老挝PIT处理详见其他Wiki页面。

## 二、境内派出机构的代扣代缴义务

### 1. 基本规则

**派出单位定义**：有外派人员且支付或负担外派人员所得的中国境内公司、企业和其他经济组织以及政府部门。

**代扣代缴规则**（财政部 税务总局公告2020年第3号）：

- 居民个人受雇于中国境内公司、企业和其他经济组织以及政府部门并派往境外工作，其所得**由境内派出单位支付或负担**的，境内派出单位为个人所得税扣缴义务人，税款由境内派出单位负责代扣代缴；
- 其所得由境外任职、受雇的**中方机构**支付、负担的，可委托其境内派出（投资）机构代征税款。

### 2. "中方机构"的定义

**中方机构**指：中国境内的公司、企业和其他经济组织以及政府部门所属的**境外分支机构、使（领）馆、子公司、代表处**等。

### 3. 工资由境外单位支付的情形处理

| 工资支付者类型 | 处理方式 |
|---|---|
| 境内派出单位支付 | 派出单位按PIT规则**预扣预缴** |
| 境外中方机构支付（子公司/分公司/代表处等） | 可由境外中方机构预扣税款，委托派出单位申报纳税 |
| 境外非中方机构支付 | 派出单位应于次年2月28日前向主管税务机关报送外派人员情况 |
| 中方机构未预扣税款 | 派出单位应于次年2月28日前向主管税务机关报送外派人员情况 |

### 4. 次年2月28日前报送的外派人员情况

派出单位需报送以下信息：

- 外派人员的姓名；
- 身份证件类型及身份证件号码；
- 职务；
- 派往国家和地区；
- 境外工作单位名称和地址；
- 派遣期限；
- 境内外收入及缴税情况。

## 三、个人境外所得年度自行申报

### 1. 申报时间

居民个人从中国境外取得所得的，应当在取得所得的**次年3月1日至6月30日内**申报纳税。

**特殊情形**：居民个人取得境外所得的境外纳税年度与公历年度不一致的，取得境外所得的境外纳税年度最后一日所在的公历年度，为境外所得对应的中国纳税年度。

### 2. 受理申报的税务机关

按照方便就近原则：

| 申报人情形 | 主管税务机关 |
|---|---|
| 有任职受雇单位 | 向任职受雇单位的主管税务机关申报 |
| 有两处及以上任职受雇单位 | 可自主选择向其中一处申报 |
| 没有任职受雇单位 | 向户籍所在地、经常居住地或者主要收入来源地的主管税务机关申报 |
| 单位代办汇算 | 向单位的主管税务机关申报 |

**主要收入来源地**：上一纳税年度向纳税人累计发放劳务报酬、稿酬及特许权使用费金额最大的扣缴义务人所在地。

### 3. 来源于中国境外的所得的识别

以下所得为来源于中国境外的所得：

1. 因任职、受雇、履约等**在中国境外提供劳务**取得的所得；
2. 中国境外企业以及其他组织支付且负担的**稿酬**所得；
3. 许可各种特许权**在中国境外使用**而取得的所得；
4. 在中国境外从事**生产、经营活动**而取得的与生产、经营活动相关的所得；
5. 从中国境外企业、其他组织以及非居民个人取得的**利息、股息、红利**所得；
6. 将财产出租给承租人**在中国境外使用**而取得的所得；
7. **转让中国境外的不动产**、转让对中国境外企业以及其他组织投资形成的股票、股权以及其他权益性资产（权益性资产）或者**在中国境外转让其他财产**取得的所得。

**重要例外**：转让对中国境外企业以及其他组织投资形成的权益性资产，该权益性资产被转让前三年（连续36个公历月份）内的任一时间，被投资企业或其他组织的资产公允价值**50%以上直接或间接来自位于中国境内的不动产**的，取得的所得**为来源于中国境内的所得**（即"间接转让中国境内不动产"规则）。

## 四、居民个人境外税收抵免

### 1. 基本原理

境内居民个人从境外取得所得，对其依照所得来源国法律**应当缴纳且实际缴纳**的所得税税额，可依法从应纳税额中抵免，但抵免额**不得超过**该纳税人境外所得依照《中华人民共和国个人所得税法》规定计算的应纳税额。

### 2. 境内外所得合并规则

| 所得类型 | 合并规则 |
|---|---|
| 综合所得（工资薪金、劳务报酬、稿酬、特许权使用费） | 境内外**合并**计算应纳税额 |
| 经营所得 | 境内外**合并**计算应纳税额（亏损不得抵减境内或他国所得，但可结转弥补同一国未来年度） |
| 其他分类所得（利息、股息、红利，财产租赁，财产转让，偶然所得） | **不与境内所得合并**，应当分别单独计算应纳税额 |

### 3. 抵免限额计算公式（分国别）

**（1）一国（地区）综合所得的抵免限额**

```
来源于一国（地区）综合所得的抵免限额
= 境内外综合所得应纳税额
  × 来源于该国（地区）的综合所得收入额
  ÷ 境内外综合所得收入额合计
```

**（2）一国（地区）经营所得的抵免限额**

```
来源于一国（地区）经营所得的抵免限额
= 境内外经营所得应纳税额
  × 来源于该国（地区）的经营所得应纳税所得额
  ÷ 境内外经营所得应纳税所得额合计
```

**（3）一国（地区）其他分类所得的抵免限额**

```
来源于一国（地区）其他分类所得的抵免限额
= 该国（地区）的其他分类所得应纳税额
```

**（4）一国（地区）所得的抵免限额**

```
来源于一国（地区）所得的抵免限额
= 综合所得抵免限额 + 经营所得抵免限额 + 其他分类所得抵免限额
```

### 4. 不可抵免的境外所得税

以下情形不包含在可抵免的境外所得税额内：

1. 按境外所得税法律属于**错缴或错征**的境外所得税税额；
2. 按税收协定（含内地与港澳税收安排）规定**不应征收**的境外所得税税额；
3. 因少缴或迟缴境外所得税而追加的**利息、滞纳金或罚款**；
4. 境外所得税纳税人或者其利害关系人从境外征税主体得到实际**返还或补偿**的境外所得税税款；
5. 按中国个人所得税法及其实施条例规定，已经**免税**的境外所得负担的境外所得税税款。

### 5. 超额部分结转

- 实际缴纳境外所得税低于抵免限额的：以实际缴纳税额作为抵免额进行抵免；
- 实际缴纳境外所得税超过抵免限额的：在限额内进行抵免，超过部分可以在**以后5个纳税年度**内结转抵免。

### 6. 饶让抵免

居民个人从与中国签订税收协定的国家（地区）取得的所得，按照该国（地区）税收法律享受免税或减税待遇，且该免税或减税的数额按照税收协定饶让条款规定**应视同已缴税额**在中国的应纳税额中抵免的，该免税或减税数额可作为居民个人实际缴纳的境外所得税税额按规定申报税收抵免。

**关键提示**：**中老税收协定无饶让抵免条款**，因此老挝PIT的减免（如基于老挝某些特殊规定的减免）不能在中国视同已缴税额抵免。

### 7. 凭证要求

- 居民个人申请抵免已在境外缴纳的个人所得税税额，应当提供**境外税务机关出具的税款所属年度的有关纳税凭证**，未提供符合要求的纳税凭证，不予抵免；
- 纳税人确实无法提供纳税凭证的，可同时凭境外所得**纳税申报表（或境外征税主体确认的缴税通知书）以及对应的银行缴款凭证**办理境外所得抵免事宜。

### 8. 追溯抵免

- 居民个人已申报境外所得、未进行税收抵免，在以后纳税年度取得纳税凭证并申报境外所得税收抵免的，可以追溯至该境外所得所属纳税年度进行抵免，但追溯年度**不得超过5年**；
- 自取得该项境外所得的5个年度内，境外征税主体出具的税款所属纳税年度纳税凭证载明的实际缴纳税额发生变化的，按实际缴纳税额重新计算并办理补退税，**不加收税收滞纳金，不退还利息**。

### 9. 汇率折算

居民个人取得来源于境外的所得或者实际已经在境外缴纳的所得税税额为**人民币以外货币**的，应当按照**办理纳税申报或者扣缴申报的上一月最后一日人民币汇率中间价**折合计算。

## 情景问答

### Q1：中资企业外派员工到老挝子公司工作，工资全部由中国母公司发放，是否需要预扣预缴个人所得税？

**答**：**需要**。居民个人被境内企业、单位、其他组织（派出单位）派往境外工作，取得的工资薪金所得或者劳务报酬所得，由派出单位或者其他境内单位支付或负担的，派出单位或者其他境内单位应按照个人所得税法及其实施条例规定预扣预缴税款。

### Q2：外派员工工资全部由老挝子公司发放，中国母公司是否有申报义务？

**答**：分两种情形：

1. **老挝子公司属于"中方机构"（即中国母公司的境外子公司）**：可以由老挝子公司预扣税款，并委托派出单位向主管税务机关申报纳税；
2. **中方机构未预扣税款的或者境外单位不是中方机构的**：派出单位应当于**次年2月28日前**向其主管税务机关报送外派人员情况，包括：外派人员的姓名、身份证件类型及身份证件号码、职务、派往国家和地区、境外工作单位名称和地址、派遣期限、境内外收入及缴税情况等。

### Q3：居民个人王某2023年度的工资薪金和境外所得情况如下，如何计算PIT？

**情景**：王某2023年取得：
- 境内工资薪金30万元，可扣除基本减除费用6万元、专项扣除6万元、专项附加扣除3.6万元、其他扣除0.24万元；
- 境外A国工资薪金20万元、特许权使用费20万元、股息4万元；
- 境外B国利息5万元；
- 在A国缴纳个人所得税8万元，在B国缴纳预提所得税0.8万元。

**答**：

**第一步：计算综合所得应纳税所得额和应纳税额**

综合所得应纳税所得额 = 30 + 20 + 20 × (1 - 20%) - 6 - 6 - 3.6 - 0.24 = **50.16万元**

（注：特许权使用费收入按80%计入综合所得收入额；股息、利息不计入综合所得）

综合所得应纳税额 = 50.16 × 30% - 5.292 = **9.756万元**

**第二步：计算境外股息、利息所得应纳税额（其他分类所得）**

- 来源于A国股息所得应纳税额 = 4 × 20% = **0.8万元**
- 来源于B国利息所得应纳税额 = 5 × 20% = **1万元**

**第三步：计算境外所得抵免限额**

- 来源于A国综合所得抵免限额 = 9.756 × (20 + 20 × 80%) ÷ (30 + 20 + 20 × 80%) = 9.756 × 36/66 ≈ **5.32万元**
- 来源于A国所得抵免限额 = 5.32（综合所得） + 0.8（股息） = **6.12万元**
- 来源于B国所得抵免限额 = **1万元**（利息）

**第四步：对比实际缴纳税额与抵免限额**

- A国实际缴纳8万元 > 抵免限额6.12万元：抵免6.12万元，超过部分（1.88万元）可结转5年内抵免；
- B国实际缴纳0.8万元 < 抵免限额1万元：抵免0.8万元（按实际缴纳抵免）。

**第五步：汇算清缴应补（退）税额**

应纳税总额 = 9.756 + 0.8 + 1 = 11.556万元
抵免税额 = 6.12 + 0.8 = 6.92万元
应补（退）税额 = 11.556 - 6.92 = **4.636万元**

（注：此例假设不考虑预缴因素；ZTL外派员工实务中应根据实际预缴情况调整。）

### Q4：外派到老挝的员工，老挝缴纳的PIT在中国汇算时是否可全额抵免？

**答**：需按**分国分项限额**判断：

1. 准备老挝税务局出具的完税凭证（纳税证明）或缴税通知书+银行付款凭证；
2. 按老挝PIT适用的六级超额累进税率（起征点1,300,000 LAK，最高税率25%）计算实际缴纳额；
3. 按中国PIT规则计算境外所得合并后的应纳税额，得出老挝抵免限额；
4. 按"孰低"原则确定实际抵免额；
5. 超过限额的部分可结转**5个年度**内抵免。

**实务提示**：老挝PIT最高税率25%通常低于中国综合所得的高档税率（最高45%），因此外派员工整体税负不会因抵免而减轻至零，通常还需在中国补税。

## 用户操作清单

### 服务中资客户的外派员工时

1. **确认派出单位身份**：
   - 核实是中国境内实体派出，还是由老挝子公司直接招聘；
   - 区分"中方机构"与非中方机构，影响中国端报送义务；
2. **协助准备中国端资料**：
   - 老挝PIT完税凭证（年度）；
   - 老挝PIT申报表（Salary Employee Form年度汇总）；
   - 银行缴款凭证；
   - 汇率折算参考（按纳税申报上月最后一日人民币中间价）；
3. **提示员工中国端申报时点**：
   - 次年3月1日至6月30日办理年度汇算清缴；
   - 境外所得可追溯抵免5年；
4. **提示饶让抵免不适用**：**中老协定无饶让抵免**，老挝PIT减免无法在中国视同已缴税抵免；
5. **派出单位合规**：
   - 若中国母公司为派出单位，需于次年2月28日前向其主管税务机关报送外派人员情况（若工资由老挝方支付）；
   - 若中国母公司直接发放工资，按月预扣预缴个税；

### 常见陷阱与提示

1. **双重征税风险**：外派员工在老挝有PIT纳税义务，中国端未进行抵免会导致双重征税；
2. **凭证取得困难**：老挝税务局出具的完税证明获取不便，可用申报表+银行缴款凭证替代；
3. **汇率折算**：使用申报上月最后一日人民币中间价，不能用年平均汇率或其他汇率；
4. **5年结转**：抵免不完的部分可结转5年，但需在当年申报时完整填报；
5. **CRS信息交换**：老挝若加入CRS，中资员工在老挝的银行账户信息可能被交换回中国税务机关，需注意合规申报全球所得。

## 备注

- 本页所述"居民个人"指在中国境内有住所，或者无住所而一个纳税年度内在中国境内居住累计满183天的个人；
- 因户籍、家庭、经济利益关系而在中国境内习惯性居住的个人属于"在中国境内有住所"；

- 对于长期驻老（如连续5年以上在老挝工作且家庭随迁）的员工，可能需要进一步分析其税收居民身份（加比规则）是否已转移至老挝；
