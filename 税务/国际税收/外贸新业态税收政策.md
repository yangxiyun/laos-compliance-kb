# 外贸新业态税收政策

## 一、概述

### 政策背景

**2021年7月**，国务院办公厅发布《关于加快发展外贸新业态模式的意见》，围绕以下6种新业态新模式提出多重支持举措：

1. **跨境电商**（跨境电子商务）；
2. **市场采购**（市场采购贸易）；
3. **外贸综合服务企业**（综服企业）；
4. **保税维修**；
5. **离岸贸易**；
6. **海外仓**。

为支持外贸新业态新模式发展，税务部门会同有关部门出台了一系列**稳外贸稳外资税收支持政策**和**征管服务便利化举措**，促进进出口平稳发展。

### 本页覆盖的7个政策（项目15-21）

| 项目 | 政策 | 核心内容 |
|---|---|---|
| 15 | 跨境电商零售出口适用退（免）税政策 | 一般纳税人+报关单+收汇+相应凭证 |
| 16 | 跨境电商零售出口免征增值税消费税 | 小规模纳税人亦可，已办税登+报关单+进货凭证 |
| 17 | 综试区零售出口无票免税 | 综试区内，无进货凭证也可免税 |
| 18 | 综试区零售出口企业所得税核定征收 | 应税所得率统一4% |
| 19 | 市场采购贸易方式出口免征增值税 | 15万美元/单以下+集聚区+商品目录 |
| 20 | 外贸综合服务企业代办退税政策 | 综服企业代国内生产企业办理 |
| 21 | 跨境电商出口退运商品税收政策 | 免征进口关税和进口环节增值税/消费税 |

## 二、跨境电商税收政策（项目15-18）

### 1. 项目15：跨境电商零售出口适用增值税、消费税退（免）税政策

#### 适用主体

符合条件的**电子商务出口企业**（自2014年1月1日起）。

**电子商务出口企业**指：

- 自建跨境电子商务销售平台的电子商务出口企业；**或**
- 利用**第三方跨境电子商务平台**开展电子商务出口的企业。

#### 政策规定

电子商务出口企业出口货物（财政部、国家税务总局明确不予出口退（免）税或免税的货物**除外**），同时符合相关条件的，适用增值税、消费税**退（免）税**政策。

#### 适用条件（4个条件，需同时满足）

1. 电子商务出口企业属于**增值税一般纳税人**并已向主管税务机关办理出口退（免）税资格认定；
2. 出口货物取得**海关出口货物报关单（出口退税专用）**，且与海关出口货物报关单电子信息一致；
3. 出口货物在**退（免）税申报期截止之日内收汇**；
4. 电子商务出口企业属于**外贸企业**的，购进出口货物取得相应的**增值税专用发票、消费税专用缴款书（分割单）或海关进口增值税、消费税专用缴款书**，且上述凭证有关内容与出口货物报关单（出口退税专用）有关内容相匹配。

### 2. 项目16：跨境电商零售出口免征增值税、消费税

#### 政策规定

自**2014年1月1日起**，电子商务出口企业出口货物，如**不符合增值税、消费税退（免）税政策条件**，但同时符合相关条件的，适用增值税、消费税**免税政策**。

#### 适用条件（3个条件，需同时满足）

1. 电子商务出口企业**已办理税务登记**；
2. 出口货物取得海关签发的**出口货物报关单**；
3. 购进出口货物取得**合法有效的进货凭证**。

#### 与项目15的区别

| 项目15（退免税） | 项目16（免税） |
|---|---|
| 增值税一般纳税人 | 可以是小规模纳税人 |
| 已办理出口退（免）税资格认定 | 仅需已办理税务登记 |
| 报关单（出口退税专用） | 报关单 |
| 需在退税申报期内收汇 | 无明确收汇要求 |
| 外贸企业需进货凭证匹配 | 需合法有效的进货凭证 |

### 3. 项目17：跨境电子商务综试区零售出口无票免税政策

#### 政策背景

自**2018年10月1日起**，对**跨境电子商务综合试验区**（综试区）内电子商务出口企业**出口未取得有效进货凭证**的货物，试行增值税、消费税**免税政策**。

#### 综试区的定义

指**经国务院批准**的跨境电子商务综合试验区。

#### 电子商务出口企业（扩大范围）

本项目中的电子商务出口企业，包括**个体工商户**（相比项目15、16扩大范围）：

- 自建跨境电子商务销售平台的企业；**或**
- 利用第三方跨境电子商务平台开展电子商务出口的**单位和个体工商户**。

#### 适用条件（3个条件，需同时满足）

1. 电子商务出口企业在**综试区注册**，并在注册地**跨境电子商务线上综合服务平台**登记出口日期、货物名称、计量单位、数量、单价、金额；
2. 出口货物通过**综试区所在地海关**办理电子商务出口申报手续；
3. 出口货物不属于财政部和税务总局根据国务院决定明确**取消出口退（免）税**的货物。

#### 核心突破

**关键点**：**不需要进货凭证**即可享受免税——这对于大量通过非正式渠道采购货源的跨境电商（如从淘宝、1688等平台进货无专票的企业）是重大利好。

### 4. 项目18：跨境电商综试区零售出口企业所得税核定征收政策

#### 政策背景

自**2020年1月1日起**，对综试区内**适用"无票免税"政策**的跨境电商企业，采用**应税所得率方式核定征收企业所得税**，应税所得率**统一按照4%**确定。

#### 适用条件（3个条件，需同时满足）

1. 在综试区注册，并在注册地跨境电子商务线上综合服务平台登记；
2. 出口货物通过综试区所在地海关办理电子商务出口申报手续；
3. 出口货物未取得有效进货凭证，其增值税、消费税享受**无票免税**政策。

#### 小型微利企业优惠叠加

综试区内实行核定征收的跨境电商企业：

- 符合**小型微利企业优惠政策**条件的，可享受小型微利企业所得税优惠政策；
- 取得的收入属于《企业所得税法》第二十六条规定的**免税收入**的，可享受免税收入优惠政策。

## 三、市场采购贸易（项目19）

### 适用主体

- 经国家批准的专业市场集聚区内的**市场经营户**；
- 自营或委托从事市场采购贸易经营的单位（**市场采购贸易经营者**）。

### 政策规定

市场经营户自营或委托市场采购贸易经营者在经国家商务主管等部门认定的市场集聚区内采购的：

- **单票报关单商品货值15万（含15万）美元以下**；
- 并在**采购地**办理出口商品通关手续；

的货物**免征增值税**。

### 试点范围（31个市场集聚区）

**浙江**（8个）：义乌国际小商品城、海宁皮革城、温州（鹿城）轻工产品交易中心、绍兴柯桥中国轻纺城、台州路桥日用品及塑料制品交易中心、湖州（织里）童装及日用消费品交易管理中心、台州路桥、义乌

**江苏**（3个）：海门叠石桥国际家纺城、常熟服装城

**广东**（5个）：广州花都皮革皮具市场、佛山亚洲国际家具材料交易中心、中山利和灯博中心、汕头宝奥国际玩具城、东莞大朗毛织贸易中心

**山东**（3个）：临沂商城工程物资市场、青岛即墨国际商贸城、烟台三站批发交易市场

**湖北**（2个）：武汉汉口北国际商品交易中心、宜昌三峡物流园

**其他省份**：河北白沟箱包市场、福建泉州石狮服装城、湖南长沙高桥大市场、四川成都国际商贸城、辽宁鞍山西柳服装城、安徽蚌埠中恒商贸城、福建晋江国际鞋纺城、河南中国（许昌）国际发制品交易市场、云南昆明俊发•新螺蛳湾国际商贸城、深圳华南国际工业原料城

**边贸商品市场**：内蒙古满洲里满购中心、广西凭祥出口商品采购中心、云南瑞丽国际商品交易市场

### 操作流程

1. 市场经营户自营出口，或与市场采购贸易经营者签订《委托代理出口货物协议》；
2. 市场经营户或市场采购贸易经营者在**市场采购贸易综合管理系统**中准确、及时录入：
   - 商品名称、规格型号、计量单位、数量、单价、金额等；
3. 形成交易清单：
   - **自营出口**：市场经营户应当于同外商签订采购合同时自行录入；
   - **委托出口**：市场经营户将货物交付市场采购贸易经营者时自行录入，或由市场采购贸易经营者录入；
4. 受托出口的市场采购贸易经营者在货物报关出口后，应在规定的期限内向主管税务机关申请开具《代理出口货物证明》；
5. 市场经营户应在**货物报关出口次月的增值税纳税申报期**内，按规定向主管税务机关办理市场采购贸易出口货物**免税申报**；
6. 委托出口的，市场采购贸易经营者可以代为办理免税申报手续。

## 四、外贸综合服务企业代办退税（项目20）

### 适用主体

- 外贸综合服务企业（**综服企业**）；
- 办理委托代办退税的生产企业。

### 政策规定

外贸综合服务企业代国内生产企业办理出口退（免）税事项，同时符合相关条件的，**可由综服企业向综服企业所在地主管税务机关集中代为办理出口退（免）税事项**（**代办退税**）。

### 综服企业的条件

1. 符合商务部等部门规定的综服企业定义并向主管税务机关**备案**；
2. 企业内部已建立**较为完善的代办退税内部风险管控制度**。

### 生产企业委托代办退税的条件（5个条件，需同时满足）

1. 出口货物为生产企业的**自产货物或视同自产货物**；
2. 生产企业为**增值税一般纳税人**并已按规定办理出口退（免）税备案；
3. 生产企业已与境外单位或个人签订**出口合同**；
4. 生产企业已与综服企业签订**外贸综合服务合同（协议）**，约定由综服企业提供包括报关报检、物流、代办退税、结算等在内的综合服务，并明确相关法律责任；
5. 生产企业向主管税务机关提供**代办退税的开户银行和账号**。

### 核心流程

1. **生产企业的销项税处理**：
   - 生产企业代办退税的出口货物，应先按**出口货物离岸价和增值税适用税率**计算销项税额并按规定**申报缴纳增值税**；
2. **向综服企业开具代办退税专用发票**：
   - 同时向综服企业开具**备注栏内注明"代办退税专用"**的增值税专用发票（**代办退税专用发票**），作为综服企业代办退税的凭证；
3. **离岸价折算**：
   - 出口货物离岸价以人民币以外的货币结算的，人民币折合率可以选择**销售额发生的当天**或**当月1日**的人民币汇率中间价；
   - 代办退税专用发票上的"金额"栏次须按照换算成人民币金额的出口货物离岸价填写；
4. **应退税额计算**：
   ```
   应退税额 = 代办退税专用发票上注明的"金额" × 出口货物适用的出口退税率
   ```
5. **代办退税专用发票的特别规定**：
   - **不得作为综服企业的增值税扣税凭证**（即综服企业不能用它抵扣自己的销项税）；
6. **备案要求**：
   - 已办理出口退（免）税备案的生产企业，在**首次委托**外贸综合服务企业代办退税前，应向其所在地主管税务机关报送**《代办退税情况备案表》**；
   - 提供代办退税账户；
   - 将与外贸综合服务企业签订的外贸综合服务合同（协议）**留存备查**。

## 五、跨境电商出口退运商品税收政策（项目21）

### 政策适用期

**2023年1月30日至2025年12月31日**期间在跨境电子商务海关监管代码（**1210、9610、9710、9810**）项下申报出口。

### 适用主体

有在跨境电子商务海关监管代码项下申报出口，因**滞销、退货**原因，自出口之日起**6个月内**原状退运进境的商品（**不含食品**）的企业。

### 政策规定

对符合条件的退运商品：

- **免征**进口关税和进口环节增值税、消费税；
- 出口时已征收的**出口关税准予退还**；
- 出口时已征收的增值税、消费税**参照内销货物发生退货**有关税收规定执行。

### 适用条件

#### 1. 时限要求

- 自出口之日起**6个月内**退运进境；
- **1210**监管代码项下出口商品：应自海关特殊监管区域或保税物流中心（B型）出区**离境之日**起6个月内退运至境内区外。

#### 2. 已办出口退税的处理

已办理出口退税的，企业应当按现行规定**补缴已退的税款**。企业应当凭主管税务机关出具的**《出口货物已补税/未退税证明》**，申请办理免征进口关税和进口环节增值税、消费税，退还出口关税手续。

#### 3. "原状退运进境"的定义

- 退运进境时的最小商品形态应与原出口时的形态**基本一致**；
- **不得增加任何配件或部件**；
- **不能经过任何加工、改装**；
- 但经拆箱、检（化）验、安装、调试等仍可视为"原状"；
- 退运进境商品应**未被使用过**；
- **例外**：只有经过试用才能发现品质不良或可证明被客户试用后退货的情况除外。

#### 4. 企业需提交的材料

- 出口商品申报清单或出口报关单；
- **退运原因说明**及证明材料：
  - **滞销退运**：企业提供"**自我声明**"作为退运原因说明材料，承诺为因滞销退运；
  - **退货退运**：企业提供**退货记录**（含跨境电子商务平台上的退货记录或拒收记录）、**返货协议**等作为退运原因说明材料。

### 《出口货物已补税/未退税证明》的开具

- 出口货物报关离境、发生退运且海关已签发出口货物报关单（出口退税专用）的，出口企业应**先**向主管税务机关申请开具**《出口货物已补税/未退税证明》**，并**携其到海关**申请办理退运手续；
- **委托出口**的货物发生退运的，应由**委托方**向主管税务机关申请开具《出口货物已补税/未退税证明》**转交受托方**，受托方凭该证明向主管税务机关申请开具《出口货物已补税/未退税证明》。

## 情景问答

### Q1：跨境电商企业签署的均是电子购销合同，是否符合备案单证规定？

**答**：**符合**。

电子购销合同是合同的一种形式。按照规定，企业可以根据自身实际，自行选择**纸质化、影像化或者数字化方式**，留存保管出口退（免）税备案单证。

### Q2：电子商务出口企业税务登记为增值税小规模纳税人，可以办理出口退税吗？

**答**：**不可以办理退税，但可以享受免税**。

**分析**：

- **项目15（退免税）**要求是增值税**一般纳税人**——小规模纳税人不适用；
- **项目16（免税）**允许已办理税务登记的企业适用——小规模纳税人可以申请。

因此，增值税小规模纳税人出口的货物**适用增值税免税政策**，但不能享受退税优惠。

### Q3：在跨境电商监管清关中心备案的跨境电子商务进出口企业，如何享受"无票免税"政策？

**答**：综试区跨境电子商务出口企业适用免税政策的零售出口货物，应按规定在**增值税、消费税纳税申报期内**向主管税务机关办理跨境电子商务零售出口货物增值税、消费税**免税申报手续**。

**关键**：**不实行免税资料备查管理**（即不需要像普通免税那样保存证明材料备查，简化管理）。

### Q4：综试区内不符合"无票免税"政策的电子商务出口企业，如何享受出口免税政策？

**答**：跨境电子商务出口企业出口货物，不符合相关出口退（免）税和"无票免税"条件，但是符合下列3种条件的，可以享受增值税、消费税免税政策（**一般免税**）：

1. 电子商务出口企业**已办理税务登记**；
2. 出口货物取得海关签发的**出口货物报关单电子信息**；
3. 购进出口货物取得**合法有效的进货凭证**。

**免税政策层级**：

```
首选：项目15（退免税，一般纳税人）
  ↓ 不符合条件
次选：项目17（综试区无票免税，综试区内且无进货凭证）
  ↓ 不符合条件
再选：项目16（一般免税，有进货凭证）
```

### Q5：跨境电商综合试验区内实行核定征收的跨境电子商务零售出口企业可以享受的企业所得税优惠政策？

**答**：

1. 符合**小型微利企业优惠政策**条件的，可享受小型微利企业所得税优惠政策；
2. 其取得的收入属于《企业所得税法》第二十六条规定的**免税收入**的，可享受免税收入优惠政策。

**应税所得率**：统一按**4%**核定。

### Q6：生产企业想委托外贸综合服务企业代办出口退税，是否需要办理委托代办退税备案？

**答**：**需要**。

已办理出口退（免）税备案的生产企业，在**首次委托**外贸综合服务企业代办退税前，应向其所在地主管税务机关报送：

- **《代办退税情况备案表》**；
- 提供代办退税账户；
- 将与外贸综合服务企业签订的外贸综合服务合同（协议）**留存备查**。

### Q7：如何开具出口货物退运已补税（未退税）证明？

**答**：

**自营出口情形**：

- 出口货物报关离境、发生退运且海关已签发出口货物报关单（出口退税专用）的，出口企业应**先**向主管税务机关申请开具《出口货物已补税/未退税证明》，并**携其到海关**申请办理退运手续。

**委托出口情形**：

- 委托出口的货物发生退运的，应由**委托方**向主管税务机关申请开具《出口货物已补税/未退税证明》**转交受托方**；
- 受托方凭该证明向主管税务机关申请开具《出口货物已补税/未退税证明》。

## 用户操作清单

### 服务中资客户时的典型场景

#### 场景1：客户涉及中国跨境电商业务向老挝销售

1. 判断是否为"跨境电商零售"：通过跨境电商平台的零售（B2C）业务；
2. 判断注册主体类型：
   - 在综试区注册的→优先考虑**项目17无票免税** + **项目18核定征收4%**；
   - 在非综试区的一般纳税人→**项目15退免税**；
   - 小规模纳税人或缺凭证的→**项目16免税**；
3. 提示客户中国端享受优惠的同时，**老挝端**需按老挝进口规则处理（进口关税+进口VAT）；
4. 对**跨境电商B2C**模式，老挝端进口的税务处理可能较简化（个人物品规则）；
5. 对**跨境电商B2B**模式，老挝端按一般进口处理。

#### 场景2：客户涉及市场采购贸易出口到老挝

1. 判断货物是否从31个试点市场集聚区采购；
2. 判断是否**单票报关单商品货值≤15万美元**；
3. 若符合：中国端免征增值税，显著降低供应链税负；
4. 提示客户：
   - 市场经营户或市场采购贸易经营者需在**市场采购贸易综合管理系统**录入交易信息；
   - 委托出口需签订《委托代理出口货物协议》。

#### 场景3：客户委托外贸综合服务企业出口到老挝

1. 判断客户是否为生产企业、出口货物是否为自产或视同自产货物；
2. 协助客户办理**委托代办退税备案**（首次委托必须备案）；
3. 关注**代办退税专用发票**的正确开具（备注栏注明"代办退税专用"）；
4. 提示：**代办退税专用发票不得作为综服企业的增值税扣税凭证**。

#### 场景4：客户在老挝销售的中国跨境电商商品被退回中国

1. 判断是否在**2023年1月30日至2025年12月31日**期间出口；
2. 判断是否在**跨境电商海关监管代码1210、9610、9710、9810**项下出口；
3. 判断是否在出口之日起**6个月内**退运进境；
4. 判断退运原因：滞销（需"自我声明"）或退货（需退货记录、返货协议）；
5. 若符合：免征进口关税和进口环节增值税、消费税；出口关税可退还；
6. 若已办理出口退税：需**补缴已退税款**并申请开具《出口货物已补税/未退税证明》。

### 与ZTL老挝业务的关联

本页政策**主要影响中国端**，对ZTL服务的**老挝端**直接关联较弱。但在以下场景有间接关联：

1. **中资企业的中老贸易架构设计**：若客户考虑将中国母公司向老挝销售产品，可对比不同模式（跨境电商、市场采购、综服企业）的中国端税负，决定最优路径；
2. **老挝子公司从中国母公司采购原材料**：若母公司使用综服企业代办退税，需关注开票流程对母子公司间定价和对老挝进口申报的影响；
3. **ZTL客户中从事老挝对华出口的企业**：本页政策不直接适用（老挝出口至中国属于中国进口方面）。

### 常见陷阱

1. **项目15/16的纳税人身份限制**：项目15要求一般纳税人，项目16可小规模；
2. **综试区无票免税与一般免税的区别**：前者不需要进货凭证，后者需要；
3. **市场采购的15万美元单票上限**：超过该金额不能享受免税；
4. **综服企业代办退税专用发票的特殊性**：不能用于综服企业自身抵扣销项税；
5. **退运商品的6个月时限**：超过6个月不能享受退运免税；
6. **退运商品的"原状"要求**：不得加装、改装、加工；
7. **已退税的处理**：退运前需先补缴已退税款，再申请免税。
