# 新加坡-老挝税收协定 - 核心条款

> 依据：Singapore–Laos DTA（2014年2月21日签订，2016年11月11日生效，2017年1月1日起执行）
> 协定文本：老挝文+英文两种，英文本为解释基准

---

## 核心要点

1. **股息双档**：持股≥10%公司股东享受**5%**；其他情形**8%**。对比中老协定"统一5%"。
2. **利息统一5%**：支付给受益所有人一律不超过5%；政府/央行/法定机构/GIC等免税。
3. **特许权使用费统一5%**：明显低于中老协定的10%（一般）/5%（设备）。
4. **服务型常设机构**：项目性服务在对方国连续或累计**12个月内超过300天**构成PE（中老协定无此条款）。
5. **建筑型PE 12个月**：与中老协定一致。
6. **资本利得来源地征税窄**：只对不动产、PE动产、国际运输动产征来源地税，**其他财产（含股权）只由居民国征税**。对比中老协定第13条有"主要由不动产构成公司股权"和"持股25%公司股权"来源地征税权——新加坡协定**没有**这两类条款。
7. **非歧视与税收优惠共存**：明确"基于国家政策的税收优惠不构成歧视"（第24条第5款）。

## 详细内容

### 一、适用税种（第2条）

| 老挝 | 新加坡 |
|---|---|
| 企业和组织的利润（所得）税 | 所得税 |
| 个人所得税 | — |

### 二、常设机构（第5条）

#### 1. 典型PE
管理地/分支机构/办事处/工厂/车间/矿井油井气井/农场种植园。

#### 2. 建筑型PE — **12个月**
建筑工地、建筑、安装或组装项目，或与之相关的监理活动，**持续12个月以上**。

#### 3. 服务型PE — **12个月内300天**
企业通过雇员或其他人员**提供服务（包括咨询服务）**，为同一或相关项目，在对方国境内**任意12个月内累计超过300天**构成PE。

> ⚠️ 这是与中老协定的重大区别——中老协定**没有服务型PE条款**，新加坡协定有。新加坡公司对老挝派驻工程师、咨询人员长期提供服务需特别注意"300天"红线。

#### 4. 代理型PE
非独立代理人在对方国以企业名义惯常签订合同构成PE。独立代理人全部/几乎全部为该企业服务且条件非独立则不算独立代理人。

### 三、股息（第10条）

| 情形 | 协定限税率 |
|---|---|
| 受益所有人为公司且**直接持股≥10%**（合伙企业除外） | **5%** |
| 其他情形 | **8%** |
| 政府/央行/GIC/法定机构等（第10条第4款）支付的股息 | **免税** |

**"政府"定义**：
- 老挝：老挝人民民主共和国政府 + 老挝人民民主共和国银行 + 地方当局 + 法定机构 + 主管当局商定的完全/主要由老挝政府所有的机构
- 新加坡：新加坡政府 + MAS（金融管理局）+ **GIC Private Limited** + 法定机构 + 主管当局商定的完全/主要由新加坡政府所有的机构

### 四、利息（第11条）

| 情形 | 协定限税率 |
|---|---|
| 受益所有人为对方居民 | **5%** |
| 支付给对方**政府**（定义同股息第4款） | **免税** |

**利息定义**：各种债权的收入（不论有无抵押担保，不论是否有权分享债务人利润），特别包括政府证券收入、债券收入（含溢价奖金）。**逾期付款罚金不视为利息**。

**反避税条款**：如因特殊关系导致利息超过独立方应商定的金额，协定只适用于独立商定部分，超额按各国国内法征税。

### 五、特许权使用费（第12条）

| 情形 | 协定限税率 |
|---|---|
| 受益所有人为对方居民 | **5%** |

**特许权使用费定义**：使用或有权使用以下作为代价的任何付款：
- 文学、艺术或科学作品版权（含**计算机软件、电影胶片、广播/电视录制的胶片磁带光盘**）
- 专利、商标、设计或模型、计划、秘密配方或工艺
- 工业、商业或科学经验的信息（know-how）

> ⚠️ 与中老协定不同，新加坡协定的特许权使用费定义**不包括"工业、商业、科学设备"的租赁**——即跨境设备租赁在新加坡协定下**不按特许权使用费处理**，而是按营业利润（第7条）处理，无PE则来源国无征税权。

### 六、资本利得（第13条）

| 转让标的 | 征税权 |
|---|---|
| 位于对方国的不动产 | 来源地（不动产所在国）可征 |
| PE营业财产动产（含PE本身转让） | 来源地（PE所在国）可征 |
| 国际运输船舶/飞机/车辆及其附属动产 | 仅居民国征税 |
| **其他财产**（含股权！） | **仅转让者居民国征税** |

> ⚠️ 这是与中老协定的重大区别：
> - 中老协定第13条有"主要由不动产构成的公司股权"和"转让前12个月持股≥25%"两类特殊条款 → 来源地可征税
> - 新加坡协定**完全没有**此类条款，**所有股权转让均只由居民国征税**
> - 这使得**通过新加坡控股架构**投资老挝子公司，股权转让回到居民国（新加坡）可争取较低甚至0税率，比中国直接持股有优势

### 七、独立个人服务（第14条）— 300天规则

对方国可征税的条件：
1. 在对方国有**固定基地**且用于开展活动；或
2. 在对方国任意12个月内**累计停留超过300天**。

对比中老协定（第十四条）：**183天或有固定基地**，新加坡协定的门槛**更宽松**（300天）。

### 八、受雇所得（第15条）— 183天/183天

非居民国受雇所得若满足三条件可继续由居民国征税：
1. 受雇人在对方国停留不超过**183天**（任意12个月内）
2. 报酬由非对方国居民雇主支付或代为支付
3. 报酬未由雇主在对方国的PE/固定基地承担

### 九、董事费（第16条）

一方居民作为对方国公司董事会成员的董事费，**可在该公司所在国征税**（即支付国征税）。

### 十、演艺人员和运动员（第17条）

演艺/运动员个人活动收入可在活动所在国征税。但若访问**完全或主要由政府公共资金支持**（政治分支/地方当局/法定机构），仅居民国征税。

### 十一、教师和研究人员（第21条）

访问对方国**公立机构**从事教学/研究**≤2年**且非私人利益的，教学/研究报酬**在对方国免税**。

### 十二、消除双重征税（第23条）

#### 老挝方
- 直接抵免：老挝居民从新加坡取得收入，老挝税前抵免在新加坡所缴税款，抵免限额按比例
- 间接抵免：老挝公司从**持股≥10%** 的新加坡公司取得股息，考虑分配利润所对应的新加坡公司税

#### 新加坡方
- 直接抵免：新加坡居民从老挝取得收入，按新加坡境外所得抵免法在新加坡税额内抵扣
- 间接抵免：**持股≥10%** 的老挝公司分派股息，考虑该公司所缴老挝税
- 抵免限额按比例

### 十三、非歧视（第24条）

- 国民待遇（第1款）、PE待遇（第2款）、企业所有权待遇（第4款）
- 第3款：不强制给予非居民国民个人免税额/减免
- **第5款特殊**：基于国家政策和标准向本国国民提供旨在促进经济或社会发展的税收优惠**不视为歧视**——这为新加坡的区域总部激励、老挝的投资促进法都留下了空间

### 十四、信息交换（第26条）

- 对比《协定》前标准的信息交换扩展条款：第5款明确**银行保密/代理持有/受益权益**不能成为拒绝提供信息的理由
- 第4款：即使对方国不需要这些信息，被要求方也须利用国内信息收集措施获取

## 协定税率速查（新加坡 vs 中老 对比）

| 项目 | 新加坡-老挝 | 中-老 |
|---|---|---|
| 股息（一般/高持股） | 8% / 5%（≥10%） | 5%（统一） |
| 利息（一般） | 5% | 10%（中→老）/ 5%（老→中）* |
| 利息（政府/政策银行） | 免税 | 免税（列名机构） |
| 特许权使用费（一般） | 5% | 10% |
| 特许权使用费（设备租赁） | **不适用**（按营业利润） | 5% |
| 服务型PE | 12个月/300天 | 无 |
| 建筑型PE | 12个月 | 12个月 |
| 受雇所得停留天数 | 183天 | 183天 |
| 独立个人服务停留天数 | 300天 | 183天（有争议） |
| 持股≥25%股权转让 | 无 — 仅居民国 | 来源地可征 |
| 饶让抵免 | 无 | 无 |

*参见本页"实务注意事项-中老协定利息条款辨析"

## 实务注意事项

1. **新加坡控股架构的股权退出优势**：通过新加坡SPV持有老挝子公司股权，未来退出时股权转让收益**只由新加坡征税**（第13条第4款），而新加坡对资本利得一般**不征税**。中国直接持股则可能被老挝按25%持股规则征来源地税。这是东南亚投资架构常用"新加坡层"的核心税务逻辑之一。
2. **新加坡不是饶让白名单国家**：即使通过新加坡架构取得老挝SEZ免税优惠，回到新加坡后由于新加坡对境外所得采用属地征税，免税留存空间大；但若最终回流中国，中国对新加坡居民企业的股息仍按25%征税（除非该新加坡公司再满足中国抵免规则），**老挝端的免税优惠不能通过饶让保留**。
3. **服务型PE 300天红线**：中资企业借道新加坡公司向老挝派遣工程师/咨询团队，单项目或系列项目累计**≤300天/12个月**才能避免在老挝构成PE被征营业利润税。须按日考勤记录。
4. **设备租赁税务定性差异**：同样是向老挝出租设备，若出租方为新加坡居民，租金按**营业利润**处理（新加坡无PE则老挝无征税权）；若出租方为中国居民，按特许权使用费**老挝征5%**。这可能带来筹划空间。
5. **股息8%非5%要注意**：SG持股<10%老挝子公司时适用8%而非5%。通过股权占比达到10%门槛较重要。
6. **信息交换条款强**：第26条第5款破除银行保密借口，新加坡-老挝之间可交换金融机构持有的账户信息，**不要假设"新加坡银行账户不可查"**。
7. **利息条款无"设备租赁特殊免税"**：新加坡协定利息条款未写明"与赊销设备有关的利息免税"条款，与某些其他国家协定不同。
8. **协定2017年1月1日起适用**：2016年11月11日前签订的合同/产生的所得不能援引本协定。早期项目需按老挝国内法或旧协定处理。

## 关键条款摘录

**第5条第3款（b）（服务型PE）**
> "缔约国一企业通过该企业为此目的雇用的雇员或其他人员提供服务，包括咨询服务，但仅限于该性质的活动（为同一或有关项目）在另一缔约国境内持续的时间在任何12个月内累计超过300天。"

**第10条第2款（股息）**
> "如果受益所有人是直接持有支付股息公司至少10%资本的公司（合伙企业除外），则征收的税款不得超过股息总额的5%；在所有其他情况下，红利总额的8%。"

**第11条第2款（利息）**
> "如果利息的实际所有人是另一缔约国的居民，则征收的税额不得超过利息总额的5%。"

**第12条第2款（特许权使用费）**
> "如果特许权使用费的实际所有人是另一缔约国的居民，则征收的税款不得超过特许权使用费总额的5%。"

**第13条第4款（其他财产转让）**
> "转让第1、第2和第3款所述财产以外的任何财产所得，应仅在转让者为其居民的缔约国纳税。"

**第24条第5款（不歧视例外）**
> "如果缔约国根据其国家政策和标准向其国民提供旨在促进经济或社会发展的税收优惠，则不应被解释为本条所指的歧视。"
