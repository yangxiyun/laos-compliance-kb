# 股息利息特许权使用费详解

## 一、概述

**股息、利息、特许权使用费**是跨国投资中最常见的三类**被动收入（Passive Income）**，也是税收协定中**"来源国有限征税权"** 的典型条款。

**三类收入的共同特点**：

1. **居民国与来源国都有征税权**（与营业利润不同）；
2. **来源国按协定限制税率**征税（通常低于国内法法定税率）；
3. **享受协定优惠税率的前提是"受益所有人"身份**；
4. **若与PE有实际联系，按营业利润处理**（常设机构优先原则）。

**与PE构成的利润处理差异**：

```
PE情况下 → 按营业利润条款（第7条）
无PE，取得被动收入 → 按股息/利息/特许权条款（第10/11/12条）
```

## 二、项目44-45：股息

### 1. 一般规则（项目44：股息优惠税率）

税收协定中股息条款一般表述：

> "如果该项股息的**受益所有人**是缔约国另一方居民，则所征税款不应超过股息总额的 \_\_%。"

**特点**：

- 居民国和来源国都有征税权；
- 来源国基于税收协定对股息所得实行**限制性税率**（一般低于来源国国内法税率）；
- 取得股息的人在**具有受益所有人身份**的情况下，才能在来源国享受协定规定的优惠税率；
- 对所支付的股息同常设机构有实际联系的**不适用股息条款**，而应按营业利润处理。

### 2. 股息协定税率速查（重要）

各国股息税率呈现3个特征：

**特征A**：个人所得税通常10%左右，企业所得税对持股25%以上的公司股东为5%（典型模式）
**特征B**：部分国家对所有股息统一5%（如老挝、埃及、黑山、克罗地亚、毛里求斯、塞尔维亚）
**特征C**：对政府、央行、主权财富基金等特殊主体免税（英国、罗马尼亚、沙特阿拉伯等）

**部分重要协定税率表**：

| 国家 | 个税税率 | 企业所得税税率 | 适用条件 |
|---|---|---|---|
| **老挝** | **5%** | **5%** | **无持股要求，统一5%** |
| 新加坡 | 10% | 5% | 持股25%以上，合伙除外 |
| 香港 | 10% | 5% | 持股25%以上 |
| 澳门 | 10% | 5% | 持股25%以上，合伙除外 |
| 越南 | 10% | — | — |
| 泰国 | 10% | — | — |
| 柬埔寨 | 10% | — | — |
| 英国 | 10% | 5% | 持股25%以上 |
| 法国 | 10% | 5% | 持股25%以上 |
| 德国 | 10% | 5% | 持股25%以上 |
| 马来西亚 | 10% | 0% | 马来西亚居民公司支付给中国居民的股息免税 |
| 美国 | 10% | — | — |

### 3. 中老协定股息条款：统一5%（独特且对ZTL客户极其重要）

**中老协定第十条**规定：

> "股息可以在缔约国另一方按照该缔约国另一方的法律征税。但是，如果收款人是股息受益所有人，则所征税款不应超过股息总额的**5%**。"

**对ZTL客户的核心意义**：

- **无持股比例要求**：不需要像中新协定那样要求持股25%以上才能享受5%；
- **无合伙企业限制**：合伙企业同样可享受；
- **个税和企税统一**：个人股东和公司股东同样按5%；
- **实际意义**：从老挝子公司汇回中国母公司的股息，老挝代扣税率为**5%**（非老挝国内法10%）；

**规划意义**：

- 中资企业将利润留在老挝子公司VS分回中国母公司：
  - 留存：免于老挝股息预提税（但可能触发CFC，见 [[受控外国企业与资本弱化]]）；
  - 分回：老挝端5%，中国端直接抵免；
- 股东贷款 vs 股权投资对比：
  - 股息：老挝5% + 中国25%（抵免5%后实际缴纳20%）；
  - 利息：老挝10%（非协定列名机构） + 中国25%（受资本弱化2:1限制）；

### 4. 项目45：股息免税（来源国免税）

对于个别国家，来源国**放弃股息所得征税权**，即中国居民取得的股息所得享受协定免税待遇。

**主要免税安排**：

| 国家/地区 | 免税条件 |
|---|---|
| 科威特、阿联酋 | 政府、政府机构、或政府全资拥有的实体；或政府直接/间接拥有至少20%股份的居民公司 |
| 沙特阿拉伯 | 政府、其所属机构或其直接/间接完全拥有的其他实体 |
| 英国 | 政府及其机构，或政府直接/间接全资所有的其他实体 |
| 格鲁吉亚 | 受益所有人直接/间接拥有支付股息公司至少50%股份，且投资超过200万欧元 |
| **马来西亚** | **马来西亚居民公司支付给中国居民的股息，在马来西亚除对公司所得征税外，免征股息税** |
| 塞浦路斯 | 塞浦路斯不对股息源泉征税时，支付给中国居民的股息无须缴纳塞浦路斯税 |
| 西班牙 | 政府、行政区、地方当局、中央银行，或政府直接/间接全资拥有的实体 |
| 新西兰 | 政府（受益所有人与其关联企业直接/间接持股不超过25%表决权） |
| 瑞士 | 政府、行政区、地方当局、双方主管当局协商同意的全资拥有机构、央行 |
| 罗马尼亚 | 政府、行政区、地方当局、政府全部或主要拥有（所有权>50%）的实体 |
| 刚果（布） | 政府、行政区、地方当局、央行、政府直接/间接全资拥有的任何机构 |
| 安哥拉 | 政府、行政区、地方当局、央行、政府主要拥有的任何机构 |

### 5. 受益所有人规则

**定义**：对所得和所得据以产生的权利或财产具有**所有权和支配权**的人，一般从事**实质性的经营活动**，可以是个人、公司或其他任何团体。

**判定原则**：按照**实质重于形式**的原则，结合具体实际情况，在综合考虑各种相关因素的基础上，进行分析判断。

**不利因素（易被否认受益所有人身份）**：

- 无支配权；
- 导管公司（Conduit Company）——壳公司、纸上公司；
- 无实际经营活动；
- 资产规模小；
- 员工数量少、无核心业务决策人员；
- 对所得所负担的税收极少或无税负。

**有利因素（支持受益所有人身份）**：

- 具有实质性经营活动；
- 资产、规模和人员配置与其职能相匹配；
- 对所得有支配权；
- 对所得承担实质风险。

## 三、项目46-47：利息

### 1. 一般规则（项目46：利息优惠税率）

利息条款一般表述：

> "如果收款人是该利息**受益所有人**，则所征税款不应超过利息总额的 \_\_%。"

**利息的定义**：

> "利息"一语是指**从各种债权取得的所得**，不论其有无抵押担保或者是否有权分享债务人的利润；特别是从公债、债券或者信用债券取得的所得，包括其溢价和奖金。**由于延期支付的罚款，不应视为利息**。

**特点**：

- 居民国和来源国都有征税权；
- 来源国基于税收协定对利息所得实行限制性税率；
- **来源国没有义务仅因利息所得是由与其缔约的另一方国家的某个居民所直接接受而放弃对该利息所得的征税权**（即需审查受益所有人）；
- 对所支付的利息同常设机构有实际联系的不适用利息条款，而应按营业利润处理。

### 2. 利息条款的特殊规定

部分协定存在特殊规定：

- **奥地利**：支付给中国银行或金融机构的利息，应仅按该利息总额的**70%**征税；
- **西班牙**：与赊销商业或科学设备有关的利息，应在首先提及的缔约国一方**免税**。

### 3. 利息协定税率常见结构

绝大多数协定采用**"一般税率10% + 列名机构免税"** 的双层结构：

| 税率层级 | 适用情形 |
|---|---|
| **0%（免税）** | 政府、行政区、地方当局；中央银行；完全为政府所有的金融机构；由缔约对方政府或机构间接提供资金/担保/保险 |
| **10%** | 其他情形（普通公司间利息） |

### 4. 中老协定利息条款：列名机构免税 + 10%一般税率

**中老协定第十一条**规定了列名机构免税和一般10%税率：

**老挝端免税的机构**（来源国在老挝、收款人为中国的以下机构）：

- 老挝人民民主共和国政府、行政区或地方当局；
- **老挝人民民主共和国银行**（老挝央行）；
- **老挝外贸银行（BCEL）**。

**中国端免税的机构**（来源国在中国、收款人为老挝的以下机构）：

- 中华人民共和国政府、行政区或地方当局；
- **中国人民银行**（中国央行）；
- **国家开发银行**；
- **中国进出口银行**；
- **中国农业发展银行**。

**其他情形**：10%一般税率。

**对ZTL客户的实务意义**：

- 中资企业（非列名机构）向老挝子公司提供贷款（或反向）的利息：老挝端10%扣缴；
- 若贷款来自中国列名银行（如国开行、进出口银行），老挝端免税——**对大型国家项目融资极为重要**；
- 中老铁路、中老经济走廊等国家战略项目，若通过中国政策性银行贷款，利息免于老挝源泉征税；

### 5. 项目47：利息免税（特定安排）

我国部分协定对特定情形下的利息所得约定在来源国免税：

- **政府间贷款**：政府、行政区、地方当局取得的利息；
- **央行间贷款**：中央银行取得的利息；
- **政策性银行贷款**：完全为政府所有的金融机构取得的利息；
- **政策性担保/保险**：由缔约对方符合条件的政府或机构提供担保/保险的贷款的利息。

## 四、项目48：特许权使用费

### 1. 特许权使用费的定义

协定中特许权使用费一般包括以下4类：

1. **文学、艺术或科学著作**（有的协定明确包括软件、电影胶片、磁带等版权）；
2. **专利、商标、设计、模型、图纸、秘密配方或秘密程序**；
3. **工业、商业、科学设备**（部分协定对此单独规定较低税率）；
4. **有关工业、商业、科学经验的信息**（know-how）。

### 2. 特许权使用费协定税率常见结构

常见的协定税率为**10%**，但部分协定对特定类别适用**5%-7%** 的优惠税率（主要针对设备租赁）：

| 国家 | 个税税率 | 企业所得税税率 | 适用范围 |
|---|---|---|---|
| **老挝** | **10%** | **—** | **一般；在老挝境内使用的为5%** |
| 新加坡 | 10% | — | 一般 |
| 新加坡 | — | 6% | 工业、商业、科学设备 |
| 越南 | 10% | — | — |
| 泰国 | 15% | — | 一般 |
| 泰国 | — | 10% | 文学、艺术或科学著作 |
| 美国 | 10% | — | 一般 |
| 德国 | 10% | — | 一般 |
| 德国 | — | 6% | 工业、商业、科学设备 |
| 印度尼西亚 | 10% | — | 版权、专利、商标等 |
| 柬埔寨 | 10% | — | 版权、专利、商标等 |

### 3. 中老协定特许权使用费条款详解（重要）

**中老协定第十二条**规定：

> "发生于缔约国一方而支付给缔约国另一方居民的特许权使用费，可以在该缔约国另一方征税。
>
> 然而，这些特许权使用费也可以在其发生的缔约国按该缔约国的法律征税。但是，如果收款人是特许权使用费受益所有人，则所征税款不应超过特许权使用费总额的**10%**。
>
> 当一方的居民因**在另一方境内使用的**设备、运输工具或类似财产而取得的所得，所征税款不应超过特许权使用费总额的**5%**。"

**关键要点**：

- **一般特许权使用费**：10%；
- **特殊情形**：在老挝境内使用的设备、运输工具或类似财产，税率降至**5%**；
- **"技术服务费"**：中老协定未单独列明技术服务费条款（与中印协定的独特条款不同）。

**对ZTL客户的实务意义**：

- 中资母公司向老挝子公司收取**专利使用费、商标许可费、技术许可费**：老挝端10%扣缴；
- 中资母公司向老挝子公司**出租机器设备**（设备在老挝境内使用）：老挝端5%扣缴；
- 实务中"技术服务费"容易与"咨询服务"混淆：
  - 若构成特许权使用费（know-how转让）→ 10%协定税率；
  - 若构成咨询服务（FWHT处理）→ 13%（10% VAT + 3% Profit Tax）——老挝国内法FWHT税率；

## 五、项目49：技术服务费（Technical Service Fee, TSF）

### 1. 概述

我国新近签订的部分税收协定**专门列有技术服务费条款**——这是OECD范本没有、但发展中国家协定范本（联合国范本）中日益重要的条款。

### 2. 列明技术服务费条款的协定

以下协定单独列明技术服务费条款：

- **印度**（独特条款，未与特许权使用费合并）；
- 部分新签协定。

### 3. 中老协定：**未单独列明技术服务费条款**

中老协定**没有专门的技术服务费条款**。因此：

- 中资企业向老挝提供技术服务取得的款项，需分析归入哪类：
  - **特许权使用费**：若涉及know-how、技术秘密、专利等 → 按第十二条10%（或设备租赁5%）；
  - **营业利润**：若构成老挝的PE → 按第七条归属PE利润在老挝征税；
  - **否则**：由居民国独占征税（中国征税，老挝不征）；
- 但**老挝国内法**对向境外企业支付的技术服务费征收**FWHT 13%**（10% VAT + 3% Profit Tax）；
- 实务中，即使协定上老挝无征税权，老挝税务机关仍会按国内法要求扣缴——此时中资企业需主动申请享受协定待遇。

### 4. 技术服务费 vs 特许权使用费的区分（重要）

| 区分维度 | 特许权使用费 | 技术服务费 |
|---|---|---|
| 核心 | **使用权的许可** | **服务的提供** |
| 客体 | 无形资产的使用权 | 人员的劳务 |
| 计价 | 按使用/销售比例支付 | 按工时/成果支付 |
| 是否独占 | 通常非独占 | 通常是独占服务 |
| 老挝协定税率 | 10%（5%对设备） | 无专门税率 |
| 老挝FWHT | — | 13% |

## 六、项目49（续）：技术服务费条款的详细展开

**中印协定**的技术服务费条款是典型代表：

> "发生于缔约国一方而支付给缔约国另一方居民的技术服务费，可以在该缔约国另一方征税。然而，这些技术服务费也可以在其发生的缔约国一方，按照该缔约国的法律征税。但是，如果收款人是技术服务费的受益所有人，则所征税款不应超过技术服务费总额的10%。"

中印协定中的"技术服务费"定义非常宽泛，几乎涵盖了**所有服务性质的跨境收入**。

中老协定**未采纳这种宽泛定义**，因此中资企业向老挝提供咨询、技术服务的所得，默认走**营业利润/FWHT**的路径。

## 情景问答

### Q1：ZTL客户（老挝中资全资子公司）向中国母公司分配股息，老挝应扣缴多少税？中国母公司如何处理？

**答**：**老挝端扣缴5%；中国母公司按25%缴企业所得税，并可就老挝已缴5%申请直接抵免**。

**老挝端**：

- 根据中老协定第十条，老挝对支付给中国居民的股息按**5%**扣缴；
- 前提：中国母公司为受益所有人（需提供COR + 受益所有人证明）；
- 与老挝国内法10%的股息WHT相比，享受协定5%优惠。

**中国母公司端**：

- 收到股息需按25%企业所得税并入应纳税所得额；
- 已在老挝缴纳的5%税款可按**直接抵免**方式抵免；
- 老挝子公司层面已缴纳的利润税（20%或SEZ税率），可按**间接抵免**方式抵免（从中国来看是间接负担的税款）。

### Q2：ZTL客户问：如果老挝子公司不分配股息，而是将利润留存老挝，对中国母公司有什么影响？

**答**：**避免了当期5%股息税 + 25%中国企业所得税，但需警惕CFC风险**。

**积极面**：

- 延迟缴税：利润留存老挝期间，老挝端免于股息WHT，中国端也不征税（未分配）；
- 再投资：留存利润可用于老挝业务扩张；

**风险面（CFC）**：

- 若老挝子公司位于SEZ享受免税，实际税负可能<12.5%；
- 若持有人为中国居民企业（通常是），且不分配非合理经营需要，可能触发CFC规则视同股息分配；
- **豁免条件**：积极经营所得、利润<500万、或位于指定非低税率国家（老挝不在此列）；
- 详见 [[受控外国企业与资本弱化]]。

### Q3：中资企业向老挝子公司提供股东贷款，利息支付涉及哪些税？

**答**：**老挝端10%扣缴 + 中国端25%企业所得税**。

**老挝端（支付方）**：

- 中老协定第十一条，老挝对支付给中国居民的利息按**10%**扣缴（除非收款方为中国央行、国开行、进出口银行、农发行等列名机构免税）；
- 利息费用在老挝子公司端能否税前扣除：
  - 需符合老挝所得税法对关联方利息扣除的限制；
  - 利率需市场化（参照独立交易原则）；
  - 资本弱化限制（老挝国内法规则 + 中国法2:1债资比）；

**中国端（收款方）**：

- 中国母公司取得老挝利息收入，按25%缴企业所得税；
- 已在老挝缴纳的10%可按直接抵免。

### Q4：中资企业向老挝子公司出租生产设备，租金按什么税率扣缴？

**答**：**老挝端5%扣缴**（中老协定优惠税率）。

**分析**：

- 中老协定第十二条第三款规定：在老挝境内使用的设备、运输工具或类似财产的租金，税率为**5%**；
- 这是中老协定中对设备租金的特殊优惠，低于一般特许权使用费的10%；
- 实务流程：
  - 中资母公司向老挝子公司开具设备租赁合同；
  - 老挝子公司支付租金时按5%扣缴（需COR + 受益所有人证明）；
  - 中资母公司取得5%扣缴凭证，在中国申请直接抵免。

### Q5：中资企业向老挝子公司收取技术许可费（含know-how和专利使用权），如何判断协定税率？

**答**：**按特许权使用费10%处理**。

**分析**：

- 技术许可费涉及**专利使用权、know-how转让**，属于中老协定第十二条规定的特许权使用费范围；
- 按一般特许权使用费10%税率扣缴；
- 若合同中明确包含**设备租金**部分，该部分按5%处理（需在合同中单独列明）；
- 注意区分**技术许可费 vs 技术服务费**：
  - 技术许可费（know-how许可）→ 特许权使用费10%；
  - 技术服务费（工程师现场指导）→ 老挝FWHT 13%（无协定优惠）。

### Q6：ZTL客户想从中国国家开发银行（国开行）借款用于老挝项目，利息扣缴税率是多少？

**答**：**免税（0%）**。

**依据**：

- 国家开发银行是中老协定第十一条第三款中列明的**中国政府全资拥有的金融机构**；
- 支付给国开行的利息在老挝端**免征所得税**；
- 其他免税的中国列名机构：中国人民银行、中国进出口银行、中国农业发展银行。

**实务意义**：

- 中老铁路、水电站等国家项目，通过中国政策性银行融资时，利息全额免老挝源泉税；
- 对比商业银行利息10%的扣缴，政策性银行贷款的税收成本显著更低；
- **规划建议**：大型基建项目融资结构设计时，优先使用政策性银行贷款。

### Q7：中资广告公司向老挝客户收取广告代理费，是否适用特许权使用费条款？

**答**：**不适用**。广告代理费属于**服务性收入**，不属于特许权使用费范畴。

**分析**：

- 特许权使用费针对知识产权的**使用权许可**（专利、商标、著作权、know-how等）；
- 广告代理是**提供服务**，不涉及使用权许可；
- 正确的税务处理：
  - 若中资公司未在老挝构成PE → 按老挝国内法FWHT扣缴13%（10% VAT + 3% Profit Tax）；
  - 若中资公司在老挝有PE → 按营业利润归属PE征税；
  - 若涉及**广告制作中产生的版权**，可能部分涉及特许权使用费（需具体分析）。

### Q8：中新税收协定中对于股息有5%、10%两档税率，中资公司在新加坡设立了一家公司，股息分回时是否可以享受5%的优惠税率？

**答**：**只有在满足一定条件的情况下才可以享受5%**。

根据中国和新加坡签订的税收协定，如果**直接拥有新加坡居民公司资本比例在取得股息前连续十二个月以内任何时候均达到至少25%**的，且为股息的受益所有人时，可以享受5%的优惠税率。

**与中老协定对比**：

- 中新协定：5% vs 10%分级；
- **中老协定：统一5%**，无持股比例要求，对中资投资者更加有利。

## 用户操作清单

### 服务中资客户时的典型场景

#### 场景1：客户规划老挝子公司股息汇回

1. 明确中老协定5%优惠税率（无持股要求）；
2. 准备**享受协定待遇**的材料：
   - 中国母公司COR（税收居民身份证明）；
   - 受益所有人身份证明（公司介绍、经营范围、员工、资产等）；
3. 协助老挝子公司在股息支付时：
   - 计算扣缴金额：5%（协定） vs 10%（国内法）；
   - 填报老挝税务申报表；
   - 保留扣缴凭证；
4. 协助中国母公司端：
   - 将老挝扣缴凭证翻译公证；
   - 提供给中国税务师办理境外所得抵免。

#### 场景2：客户考虑向老挝子公司提供股东贷款

1. 分析利息扣缴税率：
   - 中国母公司为普通企业 → 10%；
   - 中国母公司为列名机构（国开行等）→ 0%；
2. 评估资本弱化风险：
   - 中国法2:1债资比（非金融企业）；
   - 老挝国内法关联方利息扣除限制；
3. 利率合理性：
   - 参照独立交易原则（同期同类市场利率）；
   - 可参考SOFR/LIBOR + 合理点差；
4. 协助准备备案材料：
   - 贷款合同；
   - 利率合理性分析；
   - 资本弱化特殊事项文档（若超限）。

#### 场景3：客户规划设备租赁 or 技术转让

1. 区分**设备租赁** vs **技术转让**：
   - 设备租赁（在老挝使用）→ 5%；
   - 技术转让（know-how、专利）→ 10%；
   - 技术服务（工程师指导）→ 无协定优惠，按FWHT 13%处理；
2. 合同设计建议：
   - 明确区分设备、技术、服务不同组成部分；
   - 分别列明对价，便于适用不同税率；
   - 提示客户"混合合同"的税务风险；
3. 提供COR申请支持：
   - 中国母公司需取得COR以享受协定待遇。

### 常见陷阱

1. **中老协定股息统一5%，但需"受益所有人"审查**：若中间通过香港/BVI等导管公司持股，可能被否认受益所有人身份；
2. **利息10% vs 0%的关键在于是否为列名机构**：普通商业银行 ≠ 列名免税银行，需逐一核对；
3. **设备租金5%容易被误用**：仅限"在老挝境内使用"的设备，跨境设备需分析使用地；
4. **技术服务费的分类风险**：误认为"技术服务费"可享受10%特许权使用费税率——中老协定无此条款，实际按FWHT 13%处理；
5. **中间控股公司的实质性**：通过香港/新加坡SPV持股老挝子公司时，SPV需有实质性经营，否则被否认协定待遇；
6. **享受协定待遇的程序**：即使符合条件，也需主动向老挝税务机关申请享受协定待遇，提交完整材料。

## 备注

- 中老协定股息、利息、特许权使用费条款对中资企业相对友好，其中**股息5%统一税率**是中资投资老挝的重要税收优势；
- 利息免税机构的列名（中方：央行、国开行、进出口行、农发行）为政策性融资提供了税收便利；

- **中老协定无技术服务费条款**，需区分技术许可费（走特许权使用费）和技术服务费（走FWHT）；
