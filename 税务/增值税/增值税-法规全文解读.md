# 增值税法（修订版）全文解读

## 核心要点

1. **税率统一为10%** - 货物进口、国内提供商品服务、向外国法人购买商品服务均适用10%税率，出口适用0%税率
2. **扩大征税范围** - 涵盖电子系统提供的商品服务，外国法人在老挝提供的商品服务
3. **完善抵扣退税** - 明确进项税抵扣条件，优化出口退税流程，信贷有效期超过3个月可申请退税
4. **强化现代化管理** - 推行电子申报系统，通过银行系统支付，建立税收信息管理系统
5. **自2025年1月1日起施行** - 取代2018年增值税法，建立统一现代化的增值税制度

## 详细内容

### 第一章 一般规定

**立法目的**：促进投资、生产、经营、出口，增加国家预算收入，实现与地区国际接轨

**增值税定义**：根据货物进口、出口、境内提供货物服务而增加的价值征收的税

**适用范围**：在老挝开展业务和使用商品服务的国内外个人、法人、组织

**管理原则**：
- 全国统一集中管理
- 促进生产经营持续增长
- 确保服务现代、方便、公平、透明
- 与相关方协调配合
- 遵守国际公约协定

### 第二章 增值税基本制度

#### 征税活动（第十一条）
1. 从国外、经济特区进口货物
2. 在老挝境内提供商品服务
3. 外国法人在老挝提供商品服务
4. 经济特区企业向区外提供商品服务
5. 通过电子系统提供商品服务

#### 免税活动（第十二条）
**进口货物免税项目**：
- 植物动物品种、疫苗、饲料及原料
- 农业生产资料和机械
- 科研用化学品和教学设备
- 国际航空运输飞机及燃料
- 医疗器械和传统医药
- 外交机构和国际组织用品

**国内提供免税项目**：
- 初加工农产品和活畜
- 教育体育相关服务
- 金融保险服务
- 医疗健康服务
- 电力供应和出口货物

#### 税率结构（第十七条）
- **标准税率10%**：适用于所有应税货物和服务
- **零税率0%**：适用��出口货物（含成品矿产品）

#### 计税基础（第十五条）
**一般商品服务**：
- 进口：边境价格+进口税+消费税
- 国内供应：供应价值+消费税（不含增值税）
- 外国法人供应：实际购买价值（不含税费）

**矿物**：实际价格或国际市场价格或政府制定价格

**电力**：按电表记录量×合同价格或政府制定价格

#### 进项税抵扣（第二十三条）
**抵扣条件**：
1. 已获得增值税（从外国法人购买除外）
2. 通过Tax RIS系统按月申报
3. 通过银行系统支付
4. 有正确完整的扣除凭证
5. 三个月内申请扣除

**不得抵扣项目**：
- 与免税供应相关的进项税
- 非直接业务运营支出
- 个人消费和福利支出
- 无证明文件的支出
- 超过三个月未申报的进项税

### 第三章 申报缴纳

#### 申报期限（第三十条）
1. 进口货物：进口时申报
2. 一般纳税人：次月20日前申报
3. 代扣代缴：扣缴后15天内申报
4. 每次供应：实时申报

#### 缴税期限（第三十一条）
1. 进口货物：进口检查站缴纳
2. 一般纳税人：次月20日前缴纳
3. 代扣代缴：付款后15天内缴纳
4. 每次供应：实时缴纳

### 第四章 退税管理

#### 退��对象（第二十五条）
1. 出口货物的供应商
2. 连续三个月有留抵税额的纳税人
3. 解散、分立、破产企业的受让人
4. 外交机构和国际组织
5. 出境旅客和外国游客
6. 遭受重大损失的纳税人

#### 退税条件（第二十七条）
**出口企业须满足**：
- 正确记账和使用发票
- 有有效出口证明
- 通过银行系统100%收汇
- 正确完整申报纳税

## 实务注意事项

1. **电子系统申报** - 必须通过Tax RIS系统进行月度申报，建立电子档案
2. **银行支付要求** - 商品服务购销必须通过银行系统支付，现金交易无法抵扣
3. **发票管理** - 必须使用财政部印制或批准的发票，供应时必须开具
4. **三个月规则** - 进项税必须在发生月份三个月内申请抵扣，逾期只能费用化
5. **出口收汇** - 出口企业必须100%收汇才能享受退税，收汇必须通过老挝银行系统
6. **经济特区规定** - 经济特区企业向区外供应需缴纳增值税，进口可能享受优惠

## 关键条款摘录

**第二条** - 增值税是根据货物进口、货物出口、境内提供货物和服务而增加的货物和服务价值征收的税

**第十七条** - 税率：百分之十税率（10%）适用于货物进口、境内提供��品服务、向外国法人购买商品服务；零税率（0%）适用于出口货物

**第二十三条** - 进项税抵扣条件：通过Tax RIS系统申报、银行支付、正确凭证、三个月内申请

**第六十一条** - 本法自2025年1月1日起施行，取代2018年增值税法
