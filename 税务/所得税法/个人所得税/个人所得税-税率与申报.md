# 个人所得税 - 税率与申报

## 核心要点

1. PIT起征点：月收入 **1,300,000 基普**（约合人民币500元）
2. 最高税率：**25%**（月收入65,000,000基普以上）
3. 工资个税申报截止日：**次月20日前**
4. 年度汇算清缴截止日：**次年3月31日**
5. 赡养/抚养费扣除：每人每年不超过 **500万基普**，最多3人

---

## 一、有固定住所者个人所得税

### 应税收入范围（第三十四条）

1. 工资、劳务费、加班费、超时费、职务补贴、任务津贴、补助、董事会会费及其它现金或实物收益
2. 未注册企业的自由职业者收入（中介费、顾问费、服务费、施工维修服务费等）
3. 线上营销收入
4. 给股东的分红或其它收益
5. 个人或法人股份的出售或转让
6. 不通过银行和其它金融机构的借贷资本利息
7. 彩票中奖（现金或实物）
8. 价值超过130万基普的礼品、奖金
9. 土地、房屋、建筑、车辆、机械或其它资产的租赁
10. 土地、房屋、建筑或土地连带建筑物的买卖及使用权的转让
11. 知识产权（版权、专利、商标等）
12. 政府和社会组织的非经营业务
13. 遗产的继承（第三十五条18款未规定的）

### 免征所得税的收入（第三十五条）

| 免税项目 | 说明 |
|---|---|
| 工资及其他收入130万基普以下 | 起征点 |
| 使馆工作人员、国际组织工作人员、外国专家工资 | 需按协议规定 |
| 孩子未满18岁的双亲补助、生产补助、伤病补助等 | 劳动事故补助等 |
| 证券交易所登记注册的股票买卖/转让及分红 | — |
| 残疾人工资、劳务费 | 依《劳动法》或《残疾人法》 |
| 130万KIP以下的礼物、奖金 | — |
| 从账目正确完整单位取得的已缴利润税的资产租赁收入 | — |
| 国家和企业的社会保障 | — |
| 银行存款利息、企业债券、政府债券投资收入 | — |
| 人寿保险和财产保险理赔款 | — |
| 培训和继续教育费用 | — |
| 防止工伤事故的劳保服装和设备 | — |

## 三、赡养费和抚养费扣除（第五十二条）

- 允许扣除的家庭成员：父亲、母亲、夫妻、18岁以下的儿女以及没有行为能力的人
- 最多扣除 **3人**
- 每人每年不超过 **500万基普**
- 须在户口簿中体现

## 四、无固定住所者个人所得税（第三章第二节）

### 应税收入范围（第四十六条）

1. 工资、劳务费、加班费及其他收益
2. 裁判委员会、培训讲师、专家学者、艺人、运动员等的报酬
3. 中介费、咨询费
4. 彩票奖金
5. 分红、借贷利息
6. 版权费、专利权费、商标权费
7. 车辆、建筑物、机械或其他资产的买卖和租赁收入

### 无固定住所者税率（第四十九条）

| 收入类型 | 税率 |
|---|---|
| 工资、劳务费（超额累进，同有固定住所者） | 0% – 25% |
| 裁判委员会、艺人、运动员报酬 | 10% |
| 分红、借贷利息 | 10% |
| 交通工具、建筑物、机械等买卖和租赁 | 10% |
| 中介费、咨询费 | 5% |
| 培训讲师、专家学者报酬 | 5% |
| 版权费、专利权费、商标权费 | 5% |
| 彩票奖金 | 5% |

**代扣义务**：与无固定住所者签有协议的自然人、法人或组织，须在支付之日起 **15个工作日内** 申报并缴纳。

## 实务注意事项

- **工资代扣**：雇主（缴税人）有义务每月代扣代缴，发工资时直接扣减，次月20日前申报
- **无合规发票的咨询费**：属于无固定住所者服务，适用5%预提税（FWHT），支付方负责代扣
- **分红税**：外资企业向境外股东分红，适用10%预提税，需在分红之日起15个工作日内申报
- **年度汇算清缴**：次年3月31日前汇总全年收入和已缴税，多退少补
- **183天规则**：外国人在老挝工作，但在国外领取工资，若一年内连续或累计在老挝生活超过183天，须在老挝缴纳所得税
- **130万基普起征点**：月收入折合约人民币500元，在老挝外资企业员工几乎全部超标

## 关键条款摘录

> 第三十九条第1款：超额累进税率，从0%到25%，适用于本法第三十四条第1款规定。（第67号法第三十九条）

> 第五十条：与无固定住所者签有协议的自然人、法人或组织，有责任向所属税务部门或纳税人居住地税务部门计算、扣除和申报所得税，并自支付之日起15个工作日内依法准确、完整地缴纳。（第67号法第五十条）
