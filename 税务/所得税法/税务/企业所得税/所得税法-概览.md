# 老挝所得税法 - 概览

## 核心要点

1. 所得税分两大类：**企业所得税**（含利润税、小微企业所得税）和**个人所得税**
2. 本法自 **2020年1月1日** 起施行，国会字第67号，2019年6月18日通过
3. 纳税义务人范围：在老挝永久或暂时从事经营或谋生的自然人、法人及其它组织
4. 外国人在老挝工作，一年内连续或累计居住超过 **183天**，须在老挝缴纳所得税
5. 所有纳税人须依照《税收管理法》规定拥有**纳税人识别号**

## 所得税体系结构

```
所得税
├── 企业所得税
│   ├── 利润税（适用一般企业、自由职业者）
│   └── 小微企业所得税
└── 个人所得税
    ├── 有固定住所者个人所得税
    └── 无固定住所者个人所得税
```

## 重要词汇定义（第三条）

| 术语 | 定义要点 |
|---|---|
| 纳税人 | 在老挝经营或谋生，依法应缴税的自然人、法人或组织 |
| 缴税人 | 经计税、征税、扣税后依法向国家缴税的主体 |
| 有固定住所者 | 在老挝境内永久居住、生活、谋生或经营者 |
| 无固定住所者 | 在老挝境内谋生或从事经营，但不长期在老挝生活者 |
| 外国人 | 临时或长期进入老挝工作，工作到期后回国的其他国籍人士 |
| 小微企业 | 商品生产、商贸、服务行业中最小的企业，各时期有不同标准 |
| 自由职业 | 主要通过知识独立提供劳务、体力、脑力等职业化服务，如教师、律师、工程师、医师等 |

## 税务管理机构

- 财政部（最高管理机构）
- 税务司
- 省、直辖市税务厅（局）
- 县、市税务局

## 实务注意事项

- 纳税人和缴税人是两个不同概念，代扣代缴义务人（缴税人）须确保及时足额申报
- 外籍员工183天判断标准：一年内**连续或累计**在老挝生活超过183天即触发纳税义务
- 避免双重征税需依据《避免双重征税协定》（DTA）执行，老挝已与中国、新加坡等国签署DTA

## 关键条款摘录

> 第六条：因经营、就业及其它所得而须缴纳所得税的自然人、法人或其它组织，均有遵照本法缴纳所得税的义务，并须依照《税收管理法》的规定，拥有纳税人识别号。（第67号法第六条）

> 第七十六条：本法自2020年1月1日老挝人民民主共和国国家主席颁布法令实施起，登报公布15天后生效。（第67号法第七十六条）
