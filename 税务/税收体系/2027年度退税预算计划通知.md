# 2027年度退税预算计划通知

> 依据：老挝税务总局 2496/税务总局号文，2026年4月20日万象发布

---

## 核心要点

1. **提交截止 2026年5月5日**：各省/直辖市税务局、税收管理处、财产收入管理处、稽查与清算处须在此前向协调员提交2027年度退税预算计划。
2. **覆盖五大退税情形**：①0% 出口退税 ②连续3个月留抵未抵扣完的进项VAT ③旅客购物退税 ④误缴/多缴（溢缴）退税 ⑤出口商品企业退税。
3. **VAT退税计划重点三类企业**：①投资促进政策下进口环节VAT免税（特别是出口导向生产企业）；②进项留抵>3个月未抵扣完；③省级新一期道路项目（第165/NA号决议）。
4. **协调员联系方式**：显苏里德（Sengsulideth）先生，电话 020 55814141。
5. **法律依据**：第66/NA号《税收征管法》（2019）+ 第2999/MoF号决议（2021）。

---

## 详细内容

### 一、通知背景

为确保2027年度**各项税款退税工作**的执行高效、清晰、准确，并符合相关法律及下位法规定。

**发文主体**：老挝财政部 税务总局
**签发人**：税务总局局长 坎潘·潘图拉克（Khamphan Phanthoulack）
**签发日期**：2026年4月20日
**文号**：2496/税务总局

### 二、覆盖税种与退税情形

| 税种 | 退税情形 |
|---|---|
| 增值税（VAT） | ①0%税率出口退税 ②留抵3个月退税 ③旅客购物退税 ④溢缴退税 |
| 消费税 | 误缴/多缴退税、出口相关退税 |
| 利润税（企业所得税 CIT） | 误缴/多缴退税 |
| 所得税（含PIT） | 误缴/多缴退税 |
| 其他税款 | 法定退税情形 |

### 三、VAT退税计划重点（三类）

依《增值税法》（修订版）编制，重点纳入：

1. **投资促进项目免税VAT**
   - 对象：根据投资促进政策在**进口环节**享受VAT免税的项目
   - 特别强调：**出口导向型生产企业**
   - 依据：参见 [[投资促进法-概述与优惠政策]] 及 [[增值税-法规全文解读]]

2. **留抵VAT退税**
   - 对象：进项留抵税额超过**3个月未抵扣完结**的企业/单位
   - 这对应2025年新《增值税法》的"连续3个月未抵扣完可申请退税"机制

3. **省级道路项目VAT退税**
   - 对象：参与省级管辖的新一期道路项目
   - 依据：**2025年6月26日第165/NA号《国会关于批准新一期道路建设与维修促进政策的决议》**

### 四、报送要求

- **报送内容**：贵局管辖范围内个人、法人及组织的2027年度各项税款退税预测计划（详见随附表格）
- **报送方式**：汇总后发送至协调员处
- **报送截止**：**不得晚于2026年5月5日**

### 五、报送单位（受文对象）

- 全国各省、直辖市税务局局长
- 税收管理处处长
- 财产收入管理处处长
- 税务稽查与清算处处长

## 实务注意事项

1. **此通知针对税务机关内部**：不是企业直接办事指南，但透露了2027年度老挝税务总局对退税工作的**关注重点**——出口导向生产企业的进口VAT免税、留抵>3个月退税、省级道路项目。
2. **道路项目退税是新信号**：2025年第165/NA号国会决议创设了针对道路建设与维修的促进政策，2027年将进入VAT退税执行层面。做基建项目的中资客户可关注这一政策通道。
3. **留抵3个月退税是常态化机制**：新VAT法下"连续3个月未抵扣完可申请退"已成为常规退税情形，而非偶发事件。客户应建立月度进项留抵监控，超3个月主动申请。
4. **投资促进项目进口VAT免税≠全免**：注意与关税免征的区分，2025年新VAT法下临时进口加工贸易企业**仍须缴VAT**，参见 [[增值税-暂时进境税务处理]]。真正能免进口VAT的主要是投资促进目录内+出口导向型生产企业的**固定资产设备**。
5. **协调员可作第一线沟通**：显苏里德（Sengsulideth）先生是税务总局指定的此项工作协调员，若客户涉及大额退税申请，可通过省级税务局转联系。
6. **预算计划vs实际执行**：各省税局上报的是"预测"计划，最终实际退税仍按企业逐笔申请办理。但预算计划被批准后，对应企业的退税概率相对较高——有资格企业应提前与主管税局沟通被纳入。
7. **时效性强**：本通知是2026年4月发布、2027年度适用。2027年度VAT退税预算的盘子在2026年5月5日前基本定型。

## 关键条款摘录

**通知正文（VAT退税重点）**
> "关于增值税退税计划，必须严格遵守现行生效的《增值税法》（修订版）。计划中应包含：
> * 根据投资促进政策在进口环节享受增值税免税项目的增值税额，特别是出口导向型生产企业；
> * 增值税进项留抵税额超过03个月未抵扣完结的企业/单位；
> * 参与省级管辖的新一期道路项目，并根据2025年6月26日第165/NA号《国会关于批准新一期道路建设与维修促进政策的决议》享受相关政策的单位。"

**通知正文（退税税种范围）**
> "涉及的主要退税税种及情形包括：增值税（VAT）、消费税、利润税（企业所得税）、所得税及其他因适用0%退税率的出口货物、连续3个月留抵退税（未抵扣完结的进项税额）、旅客购物退税、误缴或多缴（溢缴）法定税款、出口商品企业等情形而产生的退税。"
