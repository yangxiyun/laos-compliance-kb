# 发票管理办法 - 297/总理府号政令

> 依据：老挝政府第297/总理府号政令（2022-8-29），潘坎·维帕万总理签发；颁布并在官方公报登记满15日后生效；取代2006年第12/总理府号《关于发票的政令》

---

## 核心要点

1. **2类发票3种样式**：①**纸质**（本状/套状/单张状）+ ②**电子**；样式分**增值税发票/微型企业发票/规费和服务费发票**。
2. **合法vs非法发票分界**：合法发票限定为财政部印制+企业经批印制+销售记录机开具+电子发票+特批的民众/非居民/SEZ居民买卖证明。**未经税务部门批准**的=非法发票。
3. **统一基普计价 + 老挝语首位**：发票货币原则上必须基普（除非获准外币）；语言必须老挝语，外语在右侧或下方括号内且字号更小；数字阿拉伯数字、点位千分隔、逗号小数。
4. **TaxRIS系统对接**：电子发票必须与税务部门**税务收入信息系统（TaxRIS）**相连接；商品和服务销售记录机必须允许税务部门连接。
5. **印制三方资格**：印刷厂必须有承印许可+完税证明+加入VAT系统；企业必须有营业执照+TIN+完税证明+按时纳税。
6. **保存10年**：已使用纸质发票及电子信息须**自使用之日起保存10年**。
7. **罚款体系18档**：从100,000基普（轻微违规）到承印发票价值300%（重复违规）+50,000,000基普/本（借票）+100%价值罚款（虚开发票）。
8. **跨境/出口/外币贷款项目**可申请英文发票。
9. **代办替代凭证**：民众农产品/手工艺品/林产品须**村级管理机关认证**的买卖证明；向境外非居民购买须含完整内容的合同+老挝银行系统结算。

## 详细内容

### 一、立法宗旨与适用（第1-6条）

**目的（第1条）**：规定发票管理、使用和监督检查原则，实现全国统一，确保国家预算收入征收正确、完整、及时、透明、可审计。

**发票定义（第2条）**：证明买卖商品/服务的凭证，显示商品或服务的内容、项目、数量、价值及税额。是国家用于**记账、申报、抵扣和退税、商品转移**的工具。

**关键术语（第3条）**：
| 术语 | 定义 |
|---|---|
| 销售 | 提供商品和服务、转移所有权、交换、赠送（含无偿） |
| 促销 | 抽奖、赠品等市场营销活动 |
| 销售记录机发票 | 使用销售终端、收银机等现代工具开具的纸质或电子发票 |
| **合法发票** | 财政部印制+企业经批印制+销售记录机开具+电子发票+经批的民众/非居民/SEZ居民买卖证明 |
| **非法发票** | **未经税务部门批准**的发票 |
| 规费 | 国家行政管理部门以出具证书/许可证形式收取的管理权费用 |
| 服务费 | 国家部门提供非营利服务的行政技术服务费 |
| 国内商品转移 | 商品在老挝境内不同地点之间转移 |
| 持有 | 拥有以供使用或分发给他人 |

**适用范围（第6条）**：老挝境内**商品和服务买卖双方**及与发票相关的个人、法人和组织。

### 二、发票类型、样式与内容（第7-14条）

#### 1. 类型（第7条）

| 大类 | 子类 | 说明 |
|---|---|---|
| **纸质发票** | 本状 | 印制成本，**每本≤50套**，每套**≥2联**，颜色不同、编号内容一致、带密码 |
| | 套状 | 打印机/记录机连续印制的成套发票 |
| | 单张状 | 打印机/记录机逐张印制 |
| **电子发票** | — | 电子系统记录买卖数据，可显示打印作为证明，**通过银行系统结算** |

#### 2. 样式（第8条）

| 样式 | 适用主体 |
|---|---|
| **增值税发票** | VAT系统内企业 |
| **微型企业发票** | 微型企业单位 |
| **规费和服务费发票** | 国家机关组织收取规费/服务费 |

**特殊主体**（第22条）：民众、未在老挝建企的非居民、SEZ居民→使用证明文件代替。

**尺寸**：A4、半A4、A5、小尺寸、缩微、卷筒——可选用。

#### 3. 各样式发票内容（第9条）

**增值税发票（财政部/企业自印）必备内容**：
- "增值税发票"字样
- **第1联交买方+第2联存根+第3联记账**
- 代码、编号
- 卖方：名称、企业地址、TIN、电话
- 银行存款账号
- 买方：名称、企业地址、TIN、电话
- 商品/服务序号、项目、数量、单位、单价、金额
- 总价值、VAT税率及金额、应付总金额、大写金额
- 开具日期
- 双方签名及盖章
- **印制/使用许可编号及日期**

**销售记录机开具VAT发票**：内容相似，但**无需双方签名**（机器开具特性）。

**微型企业发票**：内容与VAT发票类似，但**无VAT税率/金额**（微型企业不征VAT）。

**规费/服务费发票**：标"规费发票"或"服务费发票"，第1联给支付方、第2联存根、第3联记账。

**电子发票主要内容**（第9条第5款）：
- 发票编号
- 企业名称及地址
- 卖方TIN及电话
- 买方TIN及电话
- 开具日期
- 商品和服务项目清单
- 税种及金额
- 付款方式

> ⚠️ **关键合规点**：经税务部门认证的电子发票必须与**TaxRIS系统**连接（第9条最后段）。中资企业上电子发票系统时务必预留TaxRIS对接接口。

#### 4. 错误处理（第10条）
填写/打印发生错误或污损：**盖章作废**+**整套完整保留**。

#### 5. 编号规则（第11条）
- 编号**从小到大**顺序排列
- 多机多点开票**不得重复**代码和编号
- 销售记录机发票编号**用户不得自行设定或更改**——**只能税务部门设定和修改**

#### 6. 语言规则（第12条）
- **必须老挝语**
- 外语：**老挝语右侧或下方+括号()+字号更小**
- **可申请英文发票**的情形：
  - 向**外国游客**销售商品和服务
  - **出口**商品到国外
  - 向**外国贷款/无偿援助项目+国际组织**直接销售商品/服务

#### 7. 数字与货币（第13-14条）
- 数字：**阿拉伯数字 0-9**
- 千位分隔：**点 (.)**（如 1.000.000）
- 小数符号：**逗号 (,)**（如 1.234,56）
- 货币：**基普（LAK）**，除非获准收取外币

> ⚠️ 注意数字格式与中文/英文习惯**完全相反**——欧式格式：点为千分位、逗号为小数。中资客户开票时易出错。

### 三、发票印制与使用（第15-26条）

#### 1. 印制资格（第15-16条）

**企业单位条件**：
- 营业执照 + TIN
- 仍在有效期内的**完税证明**
- 依法按时、准确、全额履行各类纳税义务

**国家机关条件**：依法按时、准确、全额上缴规费及服务费

**印刷厂条件**：
- 营业执照 + TIN
- **加入VAT系统**
- 完税证明
- **税务部门颁发的承印发票许可**
- 依法按时纳税

#### 2. 使用规则（第17-21条）

**鼓励电子化**（第17条）：国家鼓励使用**电子发票**和**销售记录机打印的发票**。

**首次使用前公告**（第17条）：在大众媒体或网站公告 + 办公地点张贴 + 展示新发票样本。

**使用登记**（第18条）：使用前向税务部门登记发票使用，数量按需。

**分支机构与集团**（第19条）：
- 分支机构：**使用母公司发票**
- 集团公司：根据批准使用**向税务部门购买**或**自行印制**

**代理与经纪人**（第20条）：
- 独立企业代理→使用财政部发票或自印发票
- 个人/组织代理或经纪→使用所代理企业的发票

**国内商品转移**（第21条）：
- 买卖转移：**使用发票+附相关文件**证明合法性
- **无买卖的转移**（生产/进口分公司转移、省际转移、母分公司间）：必须出具**商品发货单+证明文件**

#### 3. 替代凭证（第22条）

**民众/生产组销售农产品、手工艺品、林产品、畜牧产品**：
- 持合法、内容完整的买卖证明文件
- 含双方姓名、地址、商品项目、数量、单价、金额
- **村级管理机关认证**

**国家机关出售国家财产/自然资源**：
- **国家资产管理部门**出具的买卖证明文件

**向未在老挝建企的非居民购买商品和服务**：
- 完整内容的买卖合同或证明文件
- 含双方姓名、地址、电话、商品/服务价值、VAT金额等
- 双方签字（企业还需盖章）
- **必须通过设在老挝的银行系统结算**

> ⚠️ 中资客户从中国母公司或集团关联方采购在老挝使用的服务时，若该关联方未在老挝建企（无TIN），必须按本条第22条第3款处理——**必须经老挝银行系统结算**，不能境外银行间直接划款。

#### 4. 开具时间（第24条）
- **交付/转移所有权或服务使用权时**
- 分批/分期：按每次/每期实际数量开具
- 电子发票：开具并发送给买方时

#### 5. 损坏丢失（第25条）
- 不可抗力（火灾、风暴、水灾等）导致发票损毁/丢失
- 须**事故发生或发现损失30日内**+地方行政机关证明+书面报告税务部门

#### 6. 销毁（第26条）
- 因合并/永久停业/破产/更改发票内容
- 申请销毁剩余发票
- **必须有税务官员在场监督**

### 四、企业、机关、买方权义（第27-29条）

#### 1. 企业单位/国家机关权义（第27条）
15项关键义务：
1. 按批准样式/数量/内容印制发票
2. 备**发票使用跟踪登记簿或软件**
3. 向税务部门购买/办理使用登记
4. 销售记录机/电子发票必须**与税务收入管理系统连接**
5. 每次销售/收费时开具发票
6. 依法将发票作为凭证用于**记账/申报/纳税/抵扣/退税**
7. 妥善管理保存：**已使用电子信息保存10年**
8. 允许税务部门**连接所有销售记录系统**
9. 损坏丢失须书面报告
10. 按批准要求在税务官员在场销毁
11-13. 接收信息/保密权/陈述投诉权
14. **按月、季度、年度向税务部门报告印制和使用情况**；电子发票报告须文件形式发送或U盘提交
15. 其他

#### 2. 印刷厂权义（第28条）
9项关键义务：
1. 按合同样式/数量/内容印制
2. **承印合同须经税务部门认证**（财政部委托除外）
3-4. 申请自用发票印制+开具发票
5. 印刷完成后**保留印版**，**严禁未获批使用印版印刷**
6. 在委托方+税务官员在场销毁印版和不合格品
7. **保存已使用发票或电子信息10年**
8. **按月/季度/年度报告承印情况**
9. 其他

#### 3. 买方权义（第29条）
- 每次购买/支付时**有权获取发票**
- 发票用于**转移商品、证明所有权、注册资产、抵扣税款、申请退税**
- 向卖方提供准确完整的姓名/地址等信息

### 五、禁令（第30-31条）

#### 1. 对税务人员和官员（第30条）
1. 拖延积压文件以索要报酬
2. 利用职务谋私
3. **与企业/买方串通伪造发票逃税**
4. 未经授权泄露发票信息
5. 其他

#### 2. 对企业、国家机关、印刷厂（第31条）— 15项
1. 在**未获承印许可**的印刷厂印制
2. **未经批准私自印制**发票供使用或出售
3. **国外印制**发票拿到老挝使用
4. 无许可证使用印版印制
5. **不开发票或开虚假发票**
6. 未经批准使用销售记录机或电子发票
7. **同套发票各联内容不一致**
8. **伪造发票信息逃税**
9. **非法持有发票**
10. 借出/使用他人印刷许可证
11. **借出/使用他人发票**
12. 未经批准或保存期未届满前**销毁纸质发票及电子数据**
13. **拒绝税务部门连接销售记录系统**
14. 阻挠税务官员执行职务
15. 其他

### 六、管理与检查（第32-39条）

#### 1. 管理体系（第32条）
4级架构：
1. 财政部（最高责任）
2. 税务司（财政部参谋部门）
3. 省、万象市税务局
4. 县、市、镇税务所

#### 2. 检查内容（第38条）
1. 发票相关法律和法规执行情况
2. 管理检查机构组织和运作
3. 发票印制、分配、发售和使用
4. 税务人员/官员行使权力情况

#### 3. 检查形式（第39条）
1. **常规系统检查**：定期例行
2. **事先通知检查**：必要时计划外，**提前通知**
3. **突击检查**：紧急情况，**不提前通知**

### 七、奖惩（第40-46条）

#### 1. 罚款体系（第44条）— **18档详表**

| 序号 | 违规情形 | 罚款 |
|---|---|---|
| **1** | 为未经批准企业/机关印制发票 | 第1次100% / 第2次200% / 第3次300% 承印价值 |
| **2.1** | 未经批准自印发票（**未使用**） | 第1次500,000基普/本+销毁 / 第2次1,000,000/本 |
| **2.2** | 未经批准自印发票（**已使用**） | 第1次150,000/套或7,500,000/本+全额补税 / 第2次200,000/套或10,000,000/本+补税 |
| **3** | 未获资格擅自承印 | **20,000,000基普/次** |
| **4** | 国外印制发票 | 未使用500,000/本+销毁 / 已使用200,000/套或10,000,000/本+补税 |
| **5** | 借出/使用他人印刷许可证 | **15,000,000基普/次** |
| **6** | 印制发票用于普通买卖 | 2,000,000基普/本 |
| **7** | **借出/使用他人发票** | **1,000,000基普/套或50,000,000基普/本** |
| **8** | 未按顺序编号开具 | 套状100,000/套 / 单张100,000/张 |
| **9** | 填写/打印不正确不完整 | 套状500,000/套 / 单张500,000/张 |
| **10** | 同套发票内容不一致 | **5,000,000基普/套+按最高价值追缴税款** |
| **11** | 首次使用前未公告 | 1,000,000基普 |
| **12** | **销售时不开发票** | **15,000,000基普/次+按实际销售追缴税款** |
| **13** | **虚开发票（无实际交易）** | **发票金额的100%** |
| **14** | 未经批准使用销售记录机/电子发票 | **15,000,000基普/次+追溯当年每月销售数据全额补税** |
| **15** | 弄丢发票未报告 | 整套25,000,000/套 / 整本100,000,000/本 |
| **16** | 未经批准/期限届满前销毁 | 10,000,000基普/次 |
| **17** | 非法持有发票 | 2,000,000基普/本 |
| **18** | 未报告印制和使用情况 | 100,000基普/月 或 1,200,000基普/年 |

#### 2. 刑事处罚（第45条）
构成刑事犯罪的→**《刑法》及其他刑罚法律**制裁。

#### 3. 执照中止/吊销（第46条）
税务官员可提请相关部门**暂停或吊销营业执照**。

## 实务注意事项

1. **TaxRIS系统对接是新法核心**：电子发票必须与TaxRIS连接，意味着企业开具电子发票=**实时上报税务局**。中资企业上ERP/财务系统时必须确认TaxRIS接口对接能力，否则电子发票失效=改用纸质发票（落后且不利于退税）。
2. **数字格式欧式陷阱**：点千分位+逗号小数与中国/英文习惯相反。中资客户开票系统配置时易混淆，导致金额错误从而触发第10/14条罚款。
3. **保存期10年是硬约束**：远超中国发票保存期（通常5年）。中资企业需重新设计**发票档案管理制度**，含纸质归档+电子备份+灾备方案。
4. **发票丢失30日内必须报告**：火灾/水灾后的财产清查必须在30天内完成，否则触发"丢失未报告"罚款（25,000,000/套或100,000,000/本）。中资项目应建立应急响应程序。
5. **商品发货单(DN)代发票场景**：母分公司间、生产-总部间转移用DN而非发票。中资集团内的物流调拨须严格区分"买卖转移"vs"内部转移"。
6. **向境外非居民采购必须经老挝银行结算（第22条第3款）**：中资客户向中国母公司/集团关联方付费购买服务时——必须**经老挝银行系统结算**，配套合同+VAT说明。境外汇款/银行间直接划款=违法。
7. **电子发票首次使用前的公告义务**：上线电子发票系统前须媒体/网站公告+办公地张贴样本——这是**法律强制宣传义务**，遗漏即罚款1,000,000基普。
8. **借票罚款最重**：借出/使用他人发票一档高达**50,000,000基普/本**，且累计计算。中资企业子公司之间不得"暂借票本"应急——必须各自申请。
9. **虚开发票=100%金额罚款**：在没有实际交易情况下开发票（虚开），全额罚款。这针对反避税场景，中资客户的"咨询费/管理费"等关联交易应有充分商业实质，避免被认定为虚开。
10. **印刷许可借用罚款15,000,000/次**：印刷厂层面的合规问题。中资企业若选择印刷厂时应核查其有效承印许可，避免买到非法印制的"合法格式发票"。
11. **货币原则基普但有外币例外**：第14条允许"已获准收取外币"的发票用外币。这与 [[外汇管理法-2022新版核心条款]] 第9条配合——出口、SEZ、特许项目可外币开票。中资客户应明确所属类别后选币种。
12. **2018年VAT法已被2024年第60/NA号取代**：本政令的法律依据中第48/NA号VAT法已被2024年新法替代，但**本政令本身（297号）尚未配套修订**——实务中应重点关注是否有2024年VAT新法配套发票管理修订。
13. **首次使用电子发票=系统改造工程**：上TaxRIS对接的电子发票系统涉及IT/财务/税务多部门协作，建议在年度预算中专项拨款，避免半截工程。
14. **销售记录机编号税务专控**：销售记录机生成的发票编号"用户不得自行设定或更改"——这意味着收银机上线必须由税务部门远程配置编号规则，企业不能私自调整。
15. **三层税务机关分工**：发票申请、印制许可、检查由县/省/财政部分级管辖——中资客户应了解自己归属的税务局层级，避免跨层级申请被退回。

## 关键条款摘录

**第3条第4款（合法发票定义）**
> "合法发票：指由财政部印制发行的发票、企业印制使用的发票、由商品和服务销售记录机开具的发票、电子发票，以及经税务部门批准的、未在老挝建企的民众、生产组和非居民、以及在经济特区建企的居民买卖证明文件。"

**第9条（电子发票TaxRIS连接）**
> "经税务部门认证的电子发票，必须与税务部门的税务收入信息系统（TaxRIS）相连接。"

**第11条（销售记录机编号税务专控）**
> "系统软件生成的商品和服务销售记录机发票编号，不允许用户自行设定或更改，此类编号只能由税务部门设定和修改。"

**第14条（货币原则）**
> "发票上使用的货币必须是基普（LAK），除非已获准收取外币。"

**第22条（境外采购老挝银行结算）**
> "向未在老挝建企的非居民购买商品和服务时，必须具备内容完整的买卖合同或证明文件，如：买卖双方的姓名、地址和电话号码、商品和服务的价值、增值税金额及其他必要信息，并由双方签字（若为企业还需盖章），同时必须通过设在老挝的银行系统进行结算。"

**第27条第7款（10年保存）**
> "妥善管理并保存未使用及已使用的发票以备税务官员检查时提供，已使用的电子信息需自发票使用之日起保存十年。"

**第44条第7款（借票罚款）**
> "将自己的发票借给他人使用，或使用他人发票的，将处以1,000,000基普/套 或 50,000,000基普/本 的罚款。"

**第44条第13款（虚开发票）**
> "在没有实际买卖交易的情况下开具发票（虚开发票），将处以发票填写或打印金额100%（百分之百）的罚款。"

**第48条（生效与取代）**
> "本政令自总理签字颁布，并在官方政府公报登记满十五日后生效。本政令取代2006年1月23日颁布的《关于发票的第12/总理府号政令》。"
